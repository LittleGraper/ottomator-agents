## Response Properties

<StyledTable headers={"Property,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `id` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The ID of the comment.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `owner` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The username of the user who submitted the comment.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `created` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The date and time the comment was created.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `comment` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The comment text.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `parentId` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The ID of the parent comment if it is a reply to an existing comment or `null`.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `numReplies` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The number of replies to a comment.

      <Note type={"note"}>

      A comment can only have one reply level, reply to a reply is not supported.

      </Note>

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `latestReply` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The date and time the comment's last reply in UNIX time in milliseconds, `-1` if not applicable.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>

## JSON Response Syntax

```other
{
  "id": "<comment id>",
  "owner": "<comment owner username>",
  "created": <date created shown in UNIX time>,
  "comment": "<comment text>",
  "parentId": "<parent comment id>",
  "numReplies": <num of replies>,
  "latestReply": <last reply date and time in UNIX time>
}          
```
