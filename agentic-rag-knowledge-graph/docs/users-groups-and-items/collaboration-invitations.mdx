## Example usage

The following is a sample ArcGIS Online request URL used to access the `invitations` resource:

```markdown
https://org.arcgis.com/sharing/rest/portals/0123456789ABCDEF/collaborations/9a8d5afa72444769b7bbee30184cec78/invitations?f=pjson
```

The following is a sample ArcGIS Enterprise request URL used to access the `invitations` resource:

```markdown
https://machine.domain.com/sharing/rest/portals/0123456789ABCDEF/collaborations/9a8d5afa72444769b7bbee30184cec78/invitations?f=pjson
```

## Description

The `invitations` operation returns the invitation information for all the invitations generated by a distributed collaboration host.
