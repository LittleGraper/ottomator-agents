## Example usage

The following is a sample ArcGIS Online request URL used to access the `portals` resource:

```markdown
https://org.arcgis.com/sharing/rest/portals
```

The following is a sample ArcGIS Enterprise request URL used to access the `portals` resource:

```markdown
https://machine.domain.com/webadaptor/sharing/rest/portals
```

 

## Description

The `portals` resource provides access to the [Portal Self](/portal-self) child resource, which describes this portal. The only output format supported for this resource is the default html view.
