## Example Usage

URL for Delete Items

```markdown
https://www.arcgis.com/sharing/rest/content/users/jsmith/deleteItems?items=93b54c8930ae46d9a00a7820cb3ebbd1,bb8e3d443ab44878ab8315b00b0612ca
```

## Description

Deletes a batch of items owned or administered by the calling user. 

For ArcGIS Online organizations, the `deletedItems` operation moves all the supported items to the recycle bin for 14 days before they are permanently deleted. Any [unsupported items](https://doc.arcgis.com/en/arcgis-online/reference/faq.htm#ESRI_QUESTIONANSWER_60A4B4E5F0994E8DB4A999818B19ED1D) in the batch are permanently deleted. Learn more about [permanently deleting](HTTPS://DOC.ARCGIS.COM/EN/ARCGIS-ONLINE/MANAGE-DATA/DELETE-ITEMS.HTM#ESRI_SECTION1_0B04A2F0D61D46E8AB6FA9CC4AEC68DC) and [restoring items](HTTPS://DOC.ARCGIS.COM/EN/ARCGIS-ONLINE/MANAGE-DATA/RESTORE-ITEMS.HTM).

      `permanentDelete` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Deletes a batch of items permanently. The default is `false`. Any unsupported items in the batch are permanently deleted even if the value is set to `false`. This parameter was introduced in the ArcGIS Online June 2024 update.

      Values: `true` | `false`

      Example:

      ```markdown
      https://org.arcgis.com/sharing/rest/content/users/jsmith/deleteItems?items=93b54c8930ae46d9a00a7820cb3ebbd1,bb8e3d443ab44878ab8315b00b0612ca?permanentDelete=true
      ```

      <Note type={"note"}>

      Available only for ArcGIS Online organizations. For a list of unsupported account types and items, see [here](https://doc.arcgis.com/en/arcgis-online/reference/faq.htm#ESRI_QUESTIONANSWER_60A4B4E5F0994E8DB4A999818B19ED1D).

## Response Properties

<StyledTable headers={"Property,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `results` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      An array of JSON objects, one for each item requested. Each object is made up of the `itemId` and a `success` flag to indicate if the item was successfully deleted. If an error occurs when deleting an item, an error object is also returned.

    </StyledTableCell>
  </StyledTableRow>

## JSON Response Syntax

```other
{"results": [{
  "itemId": "<item  id>",
  "success": true | false,
  "error": {
      "code": code,
      "message": "<error message>"
    }
}]
}          
```
