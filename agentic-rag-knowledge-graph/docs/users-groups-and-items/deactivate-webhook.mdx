## Example usage

The following is a sample ArcGIS Enterprise POST request for the `deactivate` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/webhooks/ef98412f3555432aa42d3aecc81c633a/deactivate HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

## Description

The `deactivate` operation pauses a specified organization webhook, which stops the webhook from delivering payloads if it's invoked. In addition to manual deactivation, webhooks will automatically deactivate when the deactivation policy requirements, set when the [webhook is created](/create-webhooks), are met. The deactivation policy can be modified using the [Update Webhook](/update-webhook) operation.
