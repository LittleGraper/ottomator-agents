## Example usage

The following is a sample ArcGIS Online POST request for the `removeParticipation` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/collaborations/eefa7ebb30aa435382cc9b27ffa95e3e/removeParticipation HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

The following is a sample ArcGIS Enterprise POST request for the `removeParticipation` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/collaborations/eefa7ebb30aa435382cc9b27ffa95e3e/removeParticipation HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

## Description

The `removeParticipation` operation removes collaboration participation by a guest from a collaboration, allowing a guest to exit a collaboration. This immediately disables further replication of data to and from the portal and the other collaboration participants.
