## Example usage

The following is a sample ArcGIS Online POST request for the `updateInfo` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/collaborations/eefa7ebb30aa435382cc9b27ffa95e3e/updateInfo HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

name=Test Collaboration&description=Test description for distributed collaboration&config={"ownerUsername":"admin"}&f=pjson

```

The following is a sample ArcGIS Enterprise POST request for the `updateInfo` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/collaborations/eefa7ebb30aa435382cc9b27ffa95e3e/updateInfo HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

name=Test Collaboration&description=Test description for distributed collaboration&config={"ownerUsername":"admin"}&f=pjson

```

## Description

The `updateInfo` operation updates certain properties of a collaboration, primarily its name, description, and configuration properties. The updates are propagated to guests when the next scheduled refresh of content occurs.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `name` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the collaboration. The collaboration name has a limit of 100 characters.

      Example

      ```markdown
      name=Test Collaboration
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `description` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The description of the collaboration. The collaboration description has a limit of 250 characters.

      Example

      ```markdown
      description=Test Description for distributed collaboration
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `config` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The configuration properties of the collaboration.

      Example

      ```markdown
      {"ownerUsername":"admin"}
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
