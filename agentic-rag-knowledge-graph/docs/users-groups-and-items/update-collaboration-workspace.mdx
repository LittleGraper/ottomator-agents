## Example usage

The following is a sample ArcGIS Online POST request for the `updateInfo` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/collaborations/3e9d1694351a4b5badef66c7599fced5/workspaces/c3c831f1274a4f228ecd87f1953c6f21/updateInfo HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

name=Test Workspace&description=Test&maxItemSizeInMB=1024&maxReplicationPackageSizeInMB=5120&f=pjson

```

The following is a sample ArcGIS Enterprise POST request for the `updateInfo` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/collaborations/3e9d1694351a4b5badef66c7599fced5/workspaces/c3c831f1274a4f228ecd87f1953c6f21/updateInfo HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

name=Test Workspace&description=Test&maxItemSizeInMB=1024&maxReplicationPackageSizeInMB=5120&f=pjson

```

## Description

The `updateInfo` operation updates certain collaboration workspace properties.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `name` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the new workspace. The workspace name has a limit of 100 characters.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `description` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The description of the new workspace. The workspace description has a limit of 250 characters.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `maxItemSizeInMB` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The maximum size (in MB) for a feature layer data item to upload and share as a copy.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `maxReplicationPackageSizeInMB` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The maximum size (in MB) for a replication package. If this limit is exceeded, no further items will be added to the replication package.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `copyByRefIfCopyFail` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Determines whether a failed attempt to copy should revert to sharing by reference. For example, in cases where the imposed size limit has been exceeded.

      Values: `true` | `false`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `config` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The configuration details of the new workspace.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
