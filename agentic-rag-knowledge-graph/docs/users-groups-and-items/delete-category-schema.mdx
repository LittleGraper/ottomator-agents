## Example usage

The following is a sample ArcGIS Online POST request for the `deleteCategorySchema` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/deleteCategorySchema HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```

Below is a sample ArcGIS Enterprise POST request for the `deleteCategorySchema` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/deleteCategorySchema HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```

## Description

The `deleteCategorySchema` operation removes the category schema set for the organization. It is only available to organization administrators.
