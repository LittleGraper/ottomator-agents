## Example usage

The following is a sample ArcGIS Online POST request for the `updateUserRole` operation:

```other
POST /sharing/rest/portals/0123456789ABCDEF/updateUserRole HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

user=swilson&role=org_publisher&f=pjson
```

The following is a sample ArcGIS Enterprise POST request for the `updateUserRole` operation:

```other
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/updateUserRole HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

user=swilson&role=org_publisher&f=pjson
```

## Description

The `updateUserRole` operation allows administrators of an organization to update the role of a user within a portal.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `role` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The role of the user to be set.

      Values: `org_admin` (for organization administrator) | `org_publisher` (for organization publisher) | `org_user` (for organization user) | `<roleId>` (for [custom roles](/role))

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `user` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The username whose role to be updated.

      Example

      ```markdown
      user=swilson
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
