## Example usage

The following is a sample ArcGIS Enterprise request URL used to access an individual server resource:

```markdown
htpps://machine.domain.com/webadaptor/sharing/rest/portals/0123456789ABCDEF/servers/J5TUOrNQurCDgHFq?f=pjson
```

## Description

The resource for an individual server represents an ArcGIS Server site that has been federated with the organization.

<Note type={"note"}>

This resource is not applicable to ArcGIS Online; it is only applicable to ArcGIS Enterprise.

</Note>

## Response properties

<StyledTable headers={"Property,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `id` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The ID of the ArcGIS Server site.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `name` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A user-provided name for the server, with a limit of 250 characters. This parameter also supports the use of double-byte characters.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `adminUrl` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The administrative URL of the ArcGIS Server site, using the fully qualified name of the machine.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `url` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The externally visible URL of the ArcGIS Server site, using the fully qualified name of the machine.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `isHosted` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A Boolean property denoting whether the ArcGIS Server site is allowed to host services for the portal (`true`) or is not allowed to host services (`false`).

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `serverKey` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The key returned when the [Register Server](/register-server) operation is called for this server. The key is only displayed once; hence, this property usually returns null.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `serverType` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The server type. 

      Values: `ArcGIS` | `ARCGIS_NOTEBOOK_SERVER` | `ARCGIS_MISSION_SERVER`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `serverRole` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The role of the server.

      Values: `STANDALONE_SERVER` | `FEDERATED_SERVER` | `FEDERATED_SERVER_WITH_RESTRICTED_PUBLISHING` | `HOSTING_SERVER`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `serverFunction` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The function of the server.

      Values: `RasterAnalytics` | `GeoEvent` | `GISServer` | `ImageHosting` | `NotebookServer` | `MissionServer` | `WorkflowManager` | `GeoAnalytics` (Deprecated at 11.4)

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `adminPublicUrl` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Introduced at 10.9.1. The administrative public URL used by federated servers to communicate with ArcGIS Enterprise on Kubernetes hosted services. This property is supported when the server role is set as `hosting_server` and an [organization-level](DEVSITE/rest/enterprise-administration/enterprise/organization-properties) `privatePortalURL` is set. This property is typically configured to use the `privatePortalURL` when ArcGIS Enterprise on Kubernetes is configured for web-tier authentication. This property is only applicable to ArcGIS Enterprise on Kubernetes deployments.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## JSON Response syntax

```other
{
  "id": "<serverId>",
  "name": "<serverName>",
  "adminUrl": "<serverAdminUrl>",
  "url": "<serverUrl>",
  "isHosted": <true|false>,
  "serverKey": "<serverKey>",
  "serverType": "<serverType>",
  "serverRole": "<serverRole>",
  "serverFunction": "<serverFunction>",
  "adminPublicUrl": "<serverAdminPublicUrl>"
}
```

## JSON Response example

The following sample response for an individual server resource is returned by an ArcGIS Enterprise:

```other
{
  "id": "J5TUOrNQurCDgHFq",
  "name": "myserver.mydomain.com:6443",
  "adminUrl": "https://myserver.mydomain.com:6443/arcgis",
  "url": "https://myserver.mydomain.com/arcgis",
  "isHosted": true,
  "serverKey": "null",
  "serverType": "ArcGIS",
  "serverRole": "HOSTING_SERVER",
  "serverFunction": "GeoAnalytics"
}
```
