
## Description

The Delete Folder Item operation (POST only) removes both the item and its link from the user's folder by default. This operation is available to the user and to the administrator of the organization to which the user belongs. 

For ArcGIS Online organizations, the `delete` operation moves the item, if supported, to the recycle bin for 14 days before it is permanently deleted. Any [unsupported items](https://doc.arcgis.com/en/arcgis-online/reference/faq.htm#ESRI_QUESTIONANSWER_60A4B4E5F0994E8DB4A999818B19ED1D) are permanently deleted. Learn more about [permanently deleting](HTTPS://DOC.ARCGIS.COM/EN/ARCGIS-ONLINE/MANAGE-DATA/DELETE-ITEMS.HTM#ESRI_SECTION1_0B04A2F0D61D46E8AB6FA9CC4AEC68DC) and [restoring items](HTTPS://DOC.ARCGIS.COM/EN/ARCGIS-ONLINE/MANAGE-DATA/RESTORE-ITEMS.HTM). 

<Note type={"note"}>

The `delete` operation has changed at ArcGIS Enterprise 10.6.1 so that when a service item that references a published service on a federated server is deleted, the corresponding service is also deleted from the server. If the item represents a capability for a service, like WMS, then the capability will be removed from its parent map service.

</Note>

  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `force` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      In edge cases where a server has been removed from a portal without unfederating, orphaned items may be left behind in the portal. In such cases, a client can pass `force = true` to force deletion of these items.

      This property was introduced at ArcGIS Enterprise 10.6.1.

      <Note type={"note"}>

      Available only for ArcGIS Enterprise deployments. 
      
      </Note>

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `permanentDelete` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Deletes an item permanently. The default is `false`. Any unsupported items are permanently deleted even if the value is set to `false`. This parameter was introduced in ArcGIS Online June 2024 update.

      Values: `true` | `false`

      Example:

      ```markdown
      https://org.arcgis.com/sharing/rest/content/users/jsmith/0bffc1c3fe984a17bd9ae705b3ca7e22/items/b512083cd1b64e2da1d3f66dbb135956/delete?permanentDelete=true
      ```

      <Note type={"note"}>

      Available only for ArcGIS Online organizations. See [here](https://doc.arcgis.com/en/arcgis-online/reference/faq.htm#ESRI_QUESTIONANSWER_60A4B4E5F0994E8DB4A999818B19ED1D) for a list of unsupported account types and items.

      </Note>

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
