## Example usage

```markdown
https://www.arcgis.com/sharing/rest/content/users/<userName>/items/<itemID>/cancel
``` 

## Description

Cancels a multipart upload on an item. Can be called after the multipart Add Item or Update Item call is made but cannot be called after a Commit operation.
