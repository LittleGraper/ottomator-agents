## Example usage

The following is a sample ArcGIS Online POST request for the `update` operation:

<Note type={"note"}>

A placeholder has been used for the `certificate` parameter's input:

</Note>

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/certificates/adzf7nGZD1XCnyph/update HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

name=orgCert&domain=domain.com&certificate=...
```

## Description

The `update` operation (HTTPS only) allows organization administrators to update a registered custom X.509 HTTPS certificate.

<Note type={"note"}>

This operation is only available for ArcGIS Online organizations.

</Note>


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `name` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Name for the registered certificate.

      Example

      ```markdown
      name=orgCert
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `domain` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Server domain that the certificate is used for.

      Example

      ```markdown
      domain=domain.com
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `certificate` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Base64-encoded certificate text, enclosed between `-----BEGIN CERTIFICATE-----` and `-----END CERTIFICATE-----`.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
