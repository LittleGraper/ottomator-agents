## Example usage

The following is a sample ArcGIS Enterprise request URL used to access an individual webhook resource:

```markdown
https://machine.domain.com/webadaptor/sharing/rest/portals/0123456789ABCDEF/webhooks/35f930b31df64b62873127613d047a1b?f=pjson
```

 

## Description

The resource for an individual wbehook. This resource provides access to the [Update Webhook](/update-webhook), [Delete Webhook](/delete-webhook), and [Deactivate Webhook](/deactivate-webhook) operations, as well as the [Notification Status](/notification-status) resource page.
