## Example usage

The following is a sample ArcGIS Online request URL used to access a user resource:

```markdown
https://www.arcgis.com/sharing/rest/community/users/jsmith?f=pjson
```

The following is a sample ArcGIS Enterprise request URL used to access a user resource:

```markdown
https://machine.domain.com/webadaptor/sharing/rest/community/users/jsmith?f=pjson
```

## Description

A user resource represents a registered user in the organization, returning details such as the user's e-mail and groups the user owns or belongs to. This resource is only accessible to the user themselves or to an organization administrator. Users who set their access setting to `private` are not visible to other organization members, except for organization administrators. A description of the user details returned by this resource are provided in the Response properties table below.

  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `groupUserMemberType` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Filters the groups returned by `groups` response property based on the group role of the user. 
      
      If `owner` is specified, only the groups that are owned by the current user are returned. If `admin` is specified, only the groups where the current user is a 'group manager' are returned. If `member` is specified, only the groups where the current user is a 'group member' are returned.
      
      Values: `owner` | `admin` | `member`

    </StyledTableCell>
