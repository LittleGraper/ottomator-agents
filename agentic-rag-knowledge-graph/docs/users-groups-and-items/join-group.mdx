## Description

Users apply to join a group using the Join Group operation (POST only). This creates a new group application, which the group administrators accept or decline. This operation also creates a notification for the user indicating that they have applied to join this group. Available only to authenticated users. 

Users can only apply to join groups to which they have access. If the group is private, users will not be able to find it to ask to join it.

Information pertaining to the applying user, such as their full name and username, can be sent as part of the group application.
