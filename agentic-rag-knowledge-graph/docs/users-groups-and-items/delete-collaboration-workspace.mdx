## Example usage

The following is a sample ArcGIS Online POST request for the `delete` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/collaborations/3e9d1694351a4b5badef66c7599fced5/workspaces/c3c831f1274a4f228ecd87f1953c6f21/delete HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

The following is a sample ArcGIS Enterprise POST request for the `delete` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/collaborations/3e9d1694351a4b5badef66c7599fced5/workspaces/c3c831f1274a4f228ecd87f1953c6f21/delete HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

## Description

The `delete` operation deletes a collaboration workspace. This immediately disables further replication of data to and from the portal and the collaboration participants.
