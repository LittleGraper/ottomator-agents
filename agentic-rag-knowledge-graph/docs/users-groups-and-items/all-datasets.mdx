## Example usage

The following is a sample ArcGIS Enterprise request URL for the `allDatasets` resource:

```markdown
https://machine.domain.com/webadaptor/sharing/rest/portals/0123456789ABCDEF/datastores/allDatasets?f=pjson
```

## Description

The `allDatasets` resource page provides access to bulk publishing operations. These operations allow users to [publish and synchronize datasets from a given data store](/publish-layers), [return a list of published layers](/get-layers), and [remove all previously published layers](/delete-layers) in preparation for the deregistration of a data store.
