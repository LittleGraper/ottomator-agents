## Example usage

The following is a sample ArcGIS Online sample POST request for the `remove` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/socialProviders/remove HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```

## Description

The `remove` operation allows organization administrators to disable social logins on an org and remove the social providers. Members who use social logins to access the org will no longer be able to use the option. This functionality only applies to ArcGIS Online.
