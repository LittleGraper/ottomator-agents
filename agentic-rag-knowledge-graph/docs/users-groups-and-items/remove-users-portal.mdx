## Example usage

The following is a sample ArcGIS Online POST request for the `removeUsers` operation:

```markdown
POST /sharing/rest/portals/01234536789ABCDEF/removeUsers HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

users=jsmith&f=pjson
```

The following is a sample ArcGIS Enterprise POST request for the `removeUsers` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/removeUsers HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

users=jsmith&f=pjson
```

## Description

The `removeUsers` operation allows the administrator to remove users from a portal. Before the administrator can remove the user, all of the user's content and groups must be reassigned or deleted.
