## Example usage

The following is a sample ArcGIS Online request URL used to access portal content:

```markdown
https://org.arcgis.com/sharing/rest/content/portals/0123456789ABCDEF
```

The following is a sample ArcGIS Enterprise request URL used to access the user content resource:

```markdown
https://machine.domain.com/webadaptor/sharing/rest/content/portals/0123456789ABCDEF
```

## Description

The portal content operation allows an administrator to return a list of all items in the organization. Only available to administrators with a privilege to view all items in the organization.

      `types` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Filter by item type. Multiple types can be filtered using commas. Item type is case sensitive. See [items and item types](/items-and-item-types) for reference.

      ```markdown
      Example: https://org.arcgis.com/sharing/rest/content/users/jsmith/1851c185888a43fcb3d6fffb78ae7c94?f=pjson&Types=Web Map,Feature Service
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `sortField` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Field to sort by.

      Supported sort field: `created` 

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `sortOrder` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Applies when working with `sortField`. Describes whether the results return in ascending or descending order. Default is ascending.

      Values: `asc` \| `desc`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `ignoreTypes` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A filter used to ignore or exclude item types that are specified by the user.

      ```markdown
      Example: https://org.arcgis.com/sharing/rest/content/users/jsmith/1851c185888a43fcb3d6fffb78ae7c94?f=pjson&ignoreTypes=Web Map,Feature Service
      ```
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `inRecycleBin` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Returns all items from the recycle bin. The default is `false`. This parameter was introduced in the ArcGIS Online June 2024 update.

      Values: `true` \| `false`

      Example:

      Returns all items in recycle bin for an organization

      ```markdown
      https://org.arcgis.com/sharing/rest/content/portals/0123456789ABCDEF?f=json&inRecycleBin=true
      ```
      
      Returns items in recycle bin for a user with filter

      ```markdown
      https://org.arcgis.com/sharing/rest/content/portals/0123456789ABCDEF?f=json&types=Web Map,Feature Service&inRecycleBin=true
      ```
      
      Returns items in recycle bin for a user with ignore filter

      ```markdown
      https://org.arcgis.com/sharing/rest/content/portals/0123456789ABCDEF?f=json&ignoreTypes=Web Map,Feature Service&inRecycleBin=true
      ```
      
      <Note type={"note"}>

      Available only for ArcGIS Online organizations. For a list of unsupported items in the recycle bin, see [here](https://doc.arcgis.com/en/arcgis-online/reference/faq.htm#ESRI_QUESTIONANSWER_60A4B4E5F0994E8DB4A999818B19ED1D).

      </Note>

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>

## Response properties

<StyledTable headers={"Property,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `total` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The total number of results found for the whole query.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `start` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The number of the first entry in the result set for this response. The index number is 1-based.

      <Note type={"note"}>

      If the `start` or `num` parameters are not provided, the default is 25 results.

      </Note>

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `num` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The number of results included in the result set for this response.

      <Note type={"note"}>

      If the `start` or `num` parameters are not provided, the default is 25 results.

      </Note>

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `nextStart` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The next entry index if the current result set doesn't contain all results, or -1 if it is the last batch.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `items` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The list of the items. See [Item](/item) for response properties for more details.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `id` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The item ID.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `title` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The item title.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `created` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The date the item was created. Shown in UNIX time.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
