## Example Usage

URL for Unshare Item

```markdown
https://www.arcgis.com/sharing/rest/content/users/jsmith/items/b512083cd1b64e2da1d3f66dbb135956/unshare
groups=4774c1c2b79046f285b2e86e5a20319e,cc5f73ab367544d6b954d82cc9c6dab7
``` 

## Description

Stops sharing the item with the specified list of groups (POST only). Available to the user and the administrator of the organization to which the user belongs, if any.
