## Example usage

The following is a sample ArcGIS Online request URL used to access the `participants` resource:

```markdown
https://org.arcgis.com/sharing/rest/portals/123456789ABCDEF/collaborations/972da76ce4a5460aaf60213279d6c3ed/participants?f=pjson
```

The following is a sample ArcGIS Enterprise request URL used to access the `participants` resource:

```markdown
https://machine.domain.com/webadaptor/sharing/rest/portals/123456789ABCDEF/collaborations/972da76ce4a5460aaf60213279d6c3ed/participants?f=pjson
```

## Description

The `participants` resource provides information about all of the participants in a distributed collaboration.
