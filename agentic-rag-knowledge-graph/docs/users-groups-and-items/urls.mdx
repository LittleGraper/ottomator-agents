## Example usage

The following is a sample ArcGIS Enterprise request URL used to access the `urls` resource:

```other
https://machine.domain.com/webadaptor/sharing/rest/portals/0123456789ABCDEF/urls?f=pjson

```


## Description

The `urls` resource returns the URL of the federated server hosting the organization's tile and feature services. This resource also provides access to operations that can be used to update the public-facing URLs for an organization's Enterprise portal and federated servers.
