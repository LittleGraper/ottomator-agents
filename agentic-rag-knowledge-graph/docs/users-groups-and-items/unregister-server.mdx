## Example usage

The following is a sample ArcGIS Enterprise POST request for the `unregister` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/servers/LJyEyWqFoczw6JUe/unregister HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

## Description

The `unregister` operation unregisters (unfederates) an ArcGIS Server site from your organization. After this operation completes, you must invoke the [Update Security Configuration](DEVSITE/rest/enterprise-administration/server/updatesecurityconfig) operation on your ArcGIS Server site to specify how you want the server to work with users and roles.
