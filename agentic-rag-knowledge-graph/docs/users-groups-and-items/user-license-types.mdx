## Example usage

Below is a sample request URL for `userLicenseType`:

```markdown
https://machine.domain.com/webadaptor/sharing/rest/community/users/jsmith/userLicenseType?f=json
``` 

## Description

This resource returns a user's assigned [license type](https://enterprise.arcgis.com/en/portal/latest/administer/windows/roles.htm). Viewing this resource through the JSON format will return information about licenses included in the user type.
