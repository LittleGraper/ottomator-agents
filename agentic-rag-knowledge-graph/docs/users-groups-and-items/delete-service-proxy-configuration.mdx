## Example usage

The following is a sample ArcGIS Enterprise POST request for the `deleteServiceProxyConfig` operation:

```markdown
POST /webadaptor/sharing/rest/portals/123456789ABCDEF/collaborations/972da76ce4a5460aaf60213279d6c3ed/workspaces/5e1aa71ba7e84c01a6620202b1e3f14a/deleteServiceProxyConfig HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```

## Description

The `deleteServiceProxyConfig` operation removes stored Viewer (Level 1) credentials saved on the collaboration workspace. These credentials allow participating portals to access services copied by reference without requiring credentials from the originating portal. Once removed, the services that were copied to other portals are updated to reflect this change.

<Note type={"note"}>

This operation is only supported for ArcGIS Enterprise-to-ArcGIS Enterprise collaborations.

</Note>
