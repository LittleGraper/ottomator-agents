## Example usage

The following is a sample ArcGIS Online POST request for the `disable` operation:

```markdown
POST /sharing/rest/community/users/NBadmin/tasks/6ea7cdac8dec466aabaa1ec258e3e601/disable HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=json
``` 

The following is a sample ArcGIS Enterprise POST request for the `disable` operation:

```markdown
POST /webadaptor/sharing/rest/community/users/NBadmin/tasks/6ea7cdac8dec466aabaa1ec258e3e601/disable HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=json
```

## Description

The `disable` operation disables a specific task, setting its `active` state to `false`.
