## Example usage

The following is a sample ArcGIS Online request URL used to access an individual item resource:

```markdown
https://www.arcgis.com/sharing/rest/content/items/10ba62fe50864339a8a3e0f18ca85506?f=pjson
``` 

The following is a sample ArcGIS Enterprise request URL used to access an individual item resource:

```markdown
https://machine.domain.com/webadaptor/sharing/rest/content/items/10ba62fe50864339a8a3e0f18ca85506?f=pjson
```

## Description

An individual item resource returns the information for a specific organization item. An item is a unit of content in the organization. Each item has a unique identifier and a well-known URL that is independent of the owner of the item. An item can have associated JSON data that's available via the [item data resource](/item-data). For example, an item of type `Map Package` returns the actual bits corresponding to the map package via the item data resource.

The `numViews` is incremented when an item is opened.
