## Example usage

The following is a sample ArcGIS Online POST request for the `createCollaboration` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/createCollaboration HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

name=Add test&description=test&workspaceName=Add test&workspaceDescription=test&portalGroupId=275e7b9c6de6490b9d63f72d3e1704ef&hostContactFirstname=Test&hostContactLastname=Admin&hostContactEmailAddress=test@domain.com&accessMode=sendAndReceive&config=&f=pjson

```

## Description

The `createCollaboration` operation creates a collaboration. The host of the collaboration is the portal where it is created. The initial workspace for the collaboration is also created. A portal group in the host portal is linked to the workspace. The access mode for the host portal is set. The contact information associated with the host can be specified; otherwise, the contact information for the administrator user performing the operation will be used.

      `name`

    <StyledTableCell cellStyle="markdown">

      The name of the collaboration. The collaboration name has a limit of 100 characters.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `description` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A description of the collaboration that all participants will see. The collaboration description has a limit of 250 characters.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `workspaceName` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the initial workspace. The workspace name has a limit of 100 characters.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `workspaceDescription` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The description of the initial workspace. The workspace description has a limit of 250 characters.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `portalGroupId` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      ID of group in the portal that will be linked with the workspace.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `accessMode` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The organization's access mode to the workspace.

      Values: `send` | `receive` | `sendAndReceive`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `hostContactFirstname` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The first name of the contact person for the collaboration host portal.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `hostContactLastname` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The last name of the contact person for the collaboration host portal.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `hostContactEmailAddress` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The email address of the contact person for the collaboration host portal.

    </StyledTableCell>
  </StyledTableRow>

    <StyledTableCell cellStyle="markdown">

    <StyledTableCell cellStyle="markdown">

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
