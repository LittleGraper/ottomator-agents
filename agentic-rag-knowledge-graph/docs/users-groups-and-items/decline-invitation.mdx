## Description

When a group administrator invites a user to their group, it results in a group invitation. The invited user can decline th e invitation using the D ecline I nvitation operation (POST only). The operation deletes the invitation and creates a notification for the user indicating that they declined the invitation. The invited user is not added to the group. Available only to authenticated users.
