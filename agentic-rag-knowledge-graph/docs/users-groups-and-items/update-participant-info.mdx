## Example usage

The following is a sample ArcGIS Online POST request for the `updateInfo` operation:

```markdown
POST /sharing/rest/portals/123456789ABCDEF/collaborations/972da76ce4a5460aaf60213279d6c3ed/participants/e6623fad-8bba-42e5-b88e-6b7e1a502ef6/updateInfo HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

contactFirstname=John&contactLastname=Smith&contactEmailAddress=jsmith@domain.com&f=pjson
```

The following is a sample ArcGIS Enterprise POST request for the `updateInfo` operation:

```markdown
POST /webadaptor/sharing/rest/portals/123456789ABCDEF/collaborations/972da76ce4a5460aaf60213279d6c3ed/participants/e6623fad-8bba-42e5-b88e-6b7e1a502ef6/updateInfo HTTP/1.1

contactFirstname=John&contactLastname=Smith&contactEmailAddress=jsmith@domain.com&f=pjson
```

## Description

The `updateInfo` operation updates the collaboration participant information for a specific participant.


##  Request parameters 

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `contactFirstName` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The first name of the contact person for the participant portal.

      Example

      ```markdown
      contactFirstName=John
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `contactLastName` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The last name of the contact person for the participant portal.

      Example

      ```markdown
      contactLastName=Smith
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `contactEmailAddress` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The email address of the contact person for the participant portal.

      Example

      ```markdown
      contactEmailAddress=jsmith@domain.com
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
