## Example usage

The following is a sample ArcGIS Enterprise POST request for the `delete` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDE/webhooks/35f930b31df64b62873127613d047a1b/delete HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

 

## Description

The `delete` operation will permanently remove the specified organization webhook from your ArcGIS Enterprise portal.
