## Example usage

The following is a sample ArcGIS Online POST request for the `refresh` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/collaborations/3e9d1694351a4b5badef66c7599fced5/invitations/fed17d86e5a44f64aed331dfba65768a/refresh HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

The following is a sample ArcGIS Enterprise POST request for the `refresh` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/collaborations/3e9d1694351a4b5badef66c7599fced5/invitations/fed17d86e5a44f64aed331dfba65768a/refresh HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

## Description

The `refresh` operation refreshes a previously generated distributed collaboration invitation. The expiration for the invitation is extended an additional 72 hours from the current time. The response for this operation is a file download the contains the collaboration invitation.
