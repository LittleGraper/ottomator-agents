## Example usage

The following is a sample ArcGIS Online POST request for the `removeResource` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/removeResource HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

key=logo.png&f=pjson
``` 

The following is a sample ArcGIS Enterprise POST request for the `removeResource` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/removeResource HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

key=logo.png&f=pjson
```

## Description

The `removeResource` operation allows the administrator to remove a file resource.
