## Description

Commit is called once all parts are uploaded during a multipart [Add Item](/add-item) or [Update Item](/update-item) operation. The parts are combined into a file, and the original file is overwritten during an [Update Item](/update-item) operation. This is an asynchronous call and returns immediately. [Status](/status) can be used to check the status of the operation until it is completed.
