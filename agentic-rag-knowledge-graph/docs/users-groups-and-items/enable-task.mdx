## Example usage

The following is a sample ArcGIS Online POST request for the `enable` operation:

```markdown
POST /sharing/rest/community/users/NBadmin/tasks/6ea7cdac8dec466aabaa1ec258e3e601/enable HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
``` 

The following is a sample ArcGIS Enterprise POST request for the `enable` operation:

```markdown
POST /webadaptor/sharing/rest/community/users/NBadmin/tasks/6ea7cdac8dec466aabaa1ec258e3e601/enable HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```

## Description

The `enable` operation enables a previously disabled task, settings its `active` state to `true`.
