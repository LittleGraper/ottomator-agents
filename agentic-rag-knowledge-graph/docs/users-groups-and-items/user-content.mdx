## Example usage

The following is a sample ArcGIS Online request URL used to access the user content resource:

```markdown
https://org.arcgis.com/sharing/rest/content/users/jsmith?start=11&num=50&f=pjson
```

The following is a sample ArcGIS Enterprise request URL used to access the user content resource:

```markdown
https://machine.domain.com/webadaptor/sharing/rest/content/users/jsmith?start=11&num=50&f=pjson
``` 

## Description

The user content resource returns information on the user's owned items. A user's content are items stored either in their root content folder (e.g. `/content/users/<username>`) or in a subfolder of the home folder with the given folder ID. Multilevel folders are not supported. Items in a folder are stored by reference and are not physically in a folder. Rather, they're stored as links to the [original item](/item) (e.g. `/content/items/<itemId>`).

<Note type={"note"}>

The endpoint is accessible only to the content's owning user or to the administrator of the user's organization.

</Note>

## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `[Common Parameters]` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      For a complete listing, see [Common parameters](/common-parameters).

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `start` 

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The number of the first entry in the result set response. The index number is 1-based. The default value of `start` is `1`.

      Example:

      ```markdown
      start=11
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `num` 

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The maximum number of results to be included in the result set response. The default value is `10`, and the maximum allowed value is `100`. Note that the actual number of returned results may be less than `num`. This happens when the number of results remaining after `start` is less than `num`.

      Example:

      ```markdown
      //Returns a max of 50 results in the response
      num=50
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `foldersContent` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Return items from all user folders. If `false`, only items from the root are returned.

      Values: `true` | `false`
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `ignoreTypes` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A filter used to ignore or exclude item types that are specified by the user.

      ```markdown
      Example: https://org.arcgis.com/sharing/rest/content/users/jsmith/1851c185888a43fcb3d6fffb78ae7c94?f=pjson&ignoreTypes=Web Map,Feature Service
      ```
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `inRecycleBin` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Returns all items from the recycle bin. The default is `false`. This parameter was introduced in the ArcGIS Online June 2024 update.

      Values: `true` \| `false`

      Example:

      Returns all items in recycle bin for a user under root folder

      ```markdown      
      https://org.arcgis.com/sharing/rest/content/users/jsmith?foldersContent=true&inRecycleBin=true
      ```
      
      Returns items in recycle bin for a user with filter

      ```markdown 
      https://org.arcgis.com/sharing/rest/content/users/jsmith?foldersContent=true&types=Web Map,Feature Service&inRecycleBin=true
      ```
      
      Returns items in recycle bin for a user with ignore filter

      ```markdown 
      https://org.arcgis.com/sharing/rest/content/users/jsmith?foldersContent=true&ignoreTypes=Web Map,Feature Service&inRecycleBin=true
      ```
      
      <Note type={"note"}>

      Available only for ArcGIS Online organizations. For a list of unsupported items in the recycle bin, see [here](https://doc.arcgis.com/en/arcgis-online/reference/faq.htm#ESRI_QUESTIONANSWER_60A4B4E5F0994E8DB4A999818B19ED1D).

      </Note>

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `types`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Filter by item type. Multiple types can be filtered using commas. Item type is case sensitive. See [items and item types](/items-and-item-types) for reference.
      
      ```markdown
      Example: https://org.arcgis.com/sharing/rest/content/users/jsmith/1851c185888a43fcb3d6fffb78ae7c94?f=pjson&Types=Web Map,Feature Service
      ```

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>

## Response properties

<StyledTable headers={"Property,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `username` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The username of the content.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `total` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The total number of results found for the whole query.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `start` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The number of the first entry in the result set for this response. The index number is 1-based.

      <Note type={"note"}>

      If the `start` or `num` parameters are not provided, the results are paginated and will display 1000 items per page.

      </Note>

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `num` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The number of results included in the result set for this response.

      <Note type={"note"}>

      If the `start` or `num` parameters are not provided, the results are paginated and will display 1000 items per page.

      </Note>

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `nextStart` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The next entry index if the current result set doesn't contain all results, or -1 if it is the last batch.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `currentFolder` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The current folder listing the user's items.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `items` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The list of the user's items. See [Item](/item) for response properties for more details.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `[folders] username` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The username of the owner of the folder.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `id` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The folder ID.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `title` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The folder title.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `created` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The date the folder was created. Shown in UNIX time.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `contentStatus` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Item content status. Organization members, as well as public members for [verified organizations](https://enterprise.arcgis.com/en/portal/latest/use/faq.htm#GUID-9958A59F-8950-4460-B866-01225AC7B3C2), can search for items that have these badges using the `contentStatus` filter. Authoritative items are automatically protected to prevent accidental deletion and are boosted in search results to increase visibility and encourage use.

      Example:

      ```markdown
      "contentStatus": "org_authoritative"
      ```

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
