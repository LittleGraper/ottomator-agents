
 

## Description

The Root resource only returns the version of the containing portal. It acts as a root to its child resources and operations. To obtain descriptive information about this portal including its name, logo, featured items, and supported protocols (HTTP vs HTTPS), access the [ Self](/portal-self) resource under [ Portals](/portals-root). Starting at ArcGIS Enterprise 11.3, the site root returns the ArcGIS Enterprise version and build numbers.
