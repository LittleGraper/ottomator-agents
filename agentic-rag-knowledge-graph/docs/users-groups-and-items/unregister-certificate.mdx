## Example usage

The following is a sample ArcGIS Online POST request for the `unregister` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/certificates/EM0lp7jNBWIYylUG/unregister HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

## Description

The `unregister` operation (HTTPS only) allows organization administrators to delete a registered certificate.

<Note type={"note"}>

This operation is only available for ArcGIS Online organizations.

</Note>
