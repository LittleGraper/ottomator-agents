## Description

<Note type={"note"}>

This operation has been deprecated in ArcGIS Online and ArcGIS Enterprise 10.7.1. Users should instead use the [Group Users List](/group-users-list) operation.

</Note>

Lists the users, owner, and administrators of a given group. Only available to members or administrators of the group.
