## Example usage

The following is a sample ArcGIS Online POST request for the `unassignUserCredits` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/unassignUserCredits HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

usernames=["user1", "user2", "user3"]&f=pjson
```

## Description

The `unassignUserCredits` operation allows organization administrators to remove credit allocation for organizational users in ArcGIS Online.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `usernames` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A JSON array of user names whose credit allocation to be removed.

      Example

      ```markdown
      usernames=["user1", "user2", "user3"]
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Response properties

<StyledTable headers={"Property,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `success` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Indicates if the operation was successful.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `notUnAssignedUsers` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      An array of user names representing a subset of users whose credit allocation were not removed successfully. Used when `success` is `true`.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
