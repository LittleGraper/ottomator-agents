## Description

When a user applies to [join](/join-group) a group, a group application is created. Group administrators can decline this application using the Decline Group Application operation (POST only). This operation deletes the application and creates a [notification](/notification) for the user indicating that the user's group application was declined. The applying user will not be added to the group. Available only to group owners and administrators.
