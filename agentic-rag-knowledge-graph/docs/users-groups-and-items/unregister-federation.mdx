## Example usage

The following is a sample ArcGIS Online POST request for the `unregister` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/idp/federation/7JgLJjJMfGXfDops/unregister HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```


## Description

The `unregister` operation allows organization administrator to remove the enterprise login set up with a SAML-based federation of identity providers.
