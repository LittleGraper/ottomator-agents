## Example Usage

```markdown
https://www.arcgis.com/sharing/rest/content/items/af01df44bf36437fa8daed01407138ab/unshare
groups=bf51aa6e879e4676b683dcbefb0ab0a9,cc5f73ab367544d6b954d82cc9c6dab7
```

## Description

Unshares a public or org item with groups that the calling user owns or administers. For the items owned by the user, this Unshare Item resource works the same as user [Unshare Item](/unshare-item-as-item-owner).
