## Example usage

The following is a sample ArcGIS Online POST request for the `unregister` operation:

```markdown
POST /sharing/rest/portals/0123456789ABCDEF/idp/GjAYMUiVDCyWuq82/unregister HTTP/1.1
Host: org.arcgis.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

The following is a sample ArcGIS Enterprise POST request for the `unregister` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/idp/GjAYMUiVDCyWuq82/unregister HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

## Description

The `unregister` operation allows organization administrator to remove the enterprise login set up with a single identity provider.
