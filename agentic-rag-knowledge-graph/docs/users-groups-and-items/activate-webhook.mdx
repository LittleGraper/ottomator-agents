## Example usage

The following is a sample ArcGIS Enterprise POST request for the `activate` operation:

```markdown
POST /webadaptor/sharing/rest/portals/0123456789ABCDEF/webhooks/ef98412f3555432aa42d3aecc81c633a/activate HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson

```

## Description

The `activate` operation restarts a deactivated organization webhook. Once a webhook is reactivated and invoked, payloads will once again be delivered to the payload URL.
