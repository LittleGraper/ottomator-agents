## Description

The `runs` resource returns all the runs that have been executed for a specific task.

## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` \| `json` \| `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>

## JSON Response syntax

```other
{
  "runs": [
    {
      "task": {
        "id": "<ID of user created task>",
        "itemId": "<Item ID of the object the task runs against>",
        "type": "<ExecuteNotebook | UpdateInsightsWorkbook>",
        "title": "<Task title>",
        "userId": "<Username of the task owner>",
        "cronSchedule": {
          "minute": "<Valid CRON expression for minute: , - * />",
          "hour": "<Valid CRON expression for hour: , - * />",
          "dayOfMonth": "<Valid CRON expression for day of month: , - * ? / L W C>",
          "month": "<Valid CRON expression for month: , - * />",
          "dayOfWeek": "<Valid CRON expression for day of the week: , - * ? / L C #>"
        },
        "runIntervalSeconds": <Length of time the run lasted>,
        "lastStart": <Date of last run expressed in Epoch milliseconds>,
        "maxOccurrences": <Maximum number of times this task will execute>,
        "parameters": {<JSON object of task parameters defined during task creation>},
        "created": <Date the task was created, expressed in Epoch milliseconds>,
        "updated": <Date the task was updated, expressed in Epoch milliseconds>,
        "startDate": <Date the task started executing, expressed in Epoch milliseconds>,
        "active": <true | false>,
        "taskState": "<active | paused | deleted | completed>"
      },
      "runId": "<ID of a specific run>",
      "url": "<Job URL from the Notebook Server Administrator API>",
      "status": "<Run status>",
      "result": "<String representation of run result>",
      "created": <Date the run was created, expressed in Epoch milliseconds>,
      "modified": <Date the run was modified, expressed in Epoch milliseconds>
    },
    ...
  ]
}
```
