## Example usage

The following is a sample ArcGIS Enterprise request URL used to access the `settings` resource:

```markdown
https://machine.domain.com/webadaptor/sharing/rest/portals/0123456789ABCDEF/webhooks/settings?f=pjson
```

 

## Description

The `settings` resource displays advanced parameters that configure the connection behavior of an organization webhook. These parameters are applied to all configured webhooks in an ArcGIS Enterprise portal and can be modified using the `update` operation.
