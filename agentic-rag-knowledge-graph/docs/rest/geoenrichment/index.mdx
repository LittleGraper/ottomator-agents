The <Term term="geoenrichment service">ArcGIS GeoEnrichment service</Term> is a <Term term="location service">location service</Term> that finds demographic information and local facts for places around the world. A point or polygon can be used to retrieve data attributes associated with the location and surrounding area. The service provides over 15,000 data fields (variables) to choose from with coverage for more than 150 countries and regions.

## Service URL 

```http
https://geoenrich.arcgis.com/arcgis/rest/services/World/geoenrichmentserver/GeoEnrichment

```
Learn more about the [service endpoint](xref:///service-endpoints).
