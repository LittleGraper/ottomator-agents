The <Term term="geocoding service">ArcGIS Geocoding service</Term> is a <Term term="location service">location service</Term> that can find addresses, businesses, and places around the world. Address text can be converted to location candidates and a location can be converted into an address. The service provides suggested address candidates for partial address and place name text. Many addresses can also be geocoded at one time with batch geocoding.
