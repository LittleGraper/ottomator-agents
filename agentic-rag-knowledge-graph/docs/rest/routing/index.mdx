The ArcGIS Routing service is a <Term term="location service">location service</Term> that can find routes, directions, and perform advanced analyses on street networks. It can solve Vehicle Routing Problem (VRP) and and last mile delivery problems that involve multiple vehicles, deliveries, and pick-ups. It also supports both <Term term="direct request">direct requests</Term> and <Term term="job request">job requests</Term> for smaller and larger processes. The service can be used to perform analyses in 240 countries using high quality street data with [global coverage](/network-coverage).
