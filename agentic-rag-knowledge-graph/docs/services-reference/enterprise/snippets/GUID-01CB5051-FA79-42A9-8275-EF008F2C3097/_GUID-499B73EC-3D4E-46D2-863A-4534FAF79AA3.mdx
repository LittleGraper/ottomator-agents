At ArcGIS Enterprise 10.9.1, named users in the Enterprise portal must be licensed with the [ArcGIS Knowledge user type extension](https://enterprise.arcgis.com/en/portal/latest/administer/windows/license-user-type-extensions.htm#ESRI_SECTION1_CC3828E6F6754E9FA670537D7C46680C) to create, edit, and update Knowledge Graph Server services. At ArcGIS Enterprise 11.0, your account no longer requires the ArcGIS Knowledge user type extension, but you must be signed in to the Enterprise portal to create, edit, and update Knowledge Graph Server services.
