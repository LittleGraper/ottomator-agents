<StyledTableCell cellStyle="markdown">

`BINARY` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Indicates that if the HRMA is less than the cut angle, the HF is set to the value associated with the zero factor; otherwise, it is set to infinity.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`FORWARD` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Establishes that only forward movement is allowed. The HRMA must be greater or equal to 0 and less than 90 degrees (0 &lt;= HRMA &lt; 90). If the HRMA is greater than 0 and less than 45 degrees, the HF for the cell is set to the value associated with the zero factor. If the HRMA is greater than or equal to 45 degrees, the side value modifier value is used. The HF for any HRMA equal to or greater than 90 degrees is set to infinity.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`LINEAR` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Specifies that the HF is a linear function of the HRMA.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`INVERSE_LAYER` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Specifies that the HF is an inverse linear function of the HRMA.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`TABLE` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

A comma-separated values (CSV) file (`.csv` ) portal item will be used to define the vertical-factor graph used to determine the HFs

</StyledTableCell>

</StyledTableRow>

</StyledTable>
