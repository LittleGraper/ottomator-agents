Analysis using `Square` or `Hexagon` bins requires a projected coordinate system. When aggregating layers into bins, the input layer or processing extent ([processSR](/process-spatial-reference) ) must have a projected coordinate system. If a projected coordinate system is not specified when running analysis, a projection will be picked based on the extent of the data.
