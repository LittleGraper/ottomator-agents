Syntax: A JSON object supports three keys: itemId, url, and uri. These keys are case sensitive. 

Example: 

```other
{"itemId": “<portal item id>”}
{“url”: “<image service url>”}
{“uri”: “<cloud raster uri or shared data path>”}

```
