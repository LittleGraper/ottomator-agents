## Description

A `Geometry Service` contains utility methods that provide access to sophisticated and frequently used geometric operations. An ArcGIS Server site can only expose one geometry service with the static name Geometry. 

<Note type={"note"}>

Geometry input and output, where required, are usually packaged as an array. See the [Geometry Objects](/geometry-objects) topic.

</Note>

Use a geometry service to do the following:

-   Buffer, project, and simplify geometry.
-   Calculate areas and lengths for geometry.
-   Determine spatial relations and label points.
-   Determine distances between geometries.
-   Apply Union, Intersection, and Difference operations between geometries.
-   Autocomplete, generalize, reshape, offset, trim or extend, and compute convex hulls of geometries.
-   Convert to or from geographic coordinate strings.

## Supported operations

The `Geometry Service` resource is primarily a processing and algorithmic resource that supports operations related to geometries.

The `Geometry Service` resource has the following operations:

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Areas and Lengths` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Calculates areas and perimeter lengths for each polygon specified in the input array.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Auto Complete` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Simplifies the process of constructing polygons that are adjacent to other polygons.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Buffer` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Returns an array of polygons at the specified distances for the input geometry. An option is available to union buffer polygons at each distance.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Convex Hull` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Returns the convex hull of the input geometry.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Cut` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Splits the input polyline or polygon where it crosses a cutting polyline.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Densify` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Densifies geometries by plotting intermediate points between existing vertices.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Difference` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Constructs the set-theoretic difference between an array of geometries and another geometry.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Distance` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Calculates the distance between two geometries.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Find Transformations` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Gets a list of geographic transformations.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`From GeoCoordinateString` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Converts a list of strings in a well-known format to x,y coordinates.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Generalize` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Returns generalized (Douglas-Peucker) versions of the input geometries.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Intersect` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Constructs the set-theoretic intersection between an array of geometries and another geometry.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Label Points` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Calculates an interior point for each polygon specified in the input array. 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Lengths` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Calculates the length of each polyline specified in the input array.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Offset` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Constructs the offset of the given input polyline based on an offset distance.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Project` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Returns an array of projected geometries.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Relation` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Determines the pairs of geometries from the input geometry arrays that participate in the specified spatial relation.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Reshape` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Reshapes a polyline or a part of a polygon using a reshaping line.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Simplify` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Returns an array of topologically correct geometries.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`To GeoCoordinateString` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Converts a list of coordinates into strings in a well-known format based on the specified conversion type.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Trim/Extend` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Trims or extends each polyline specified in the input array using the user-specified guide polylines.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`Union` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Constructs the set-theoretic union of the input geometries.

</StyledTableCell>

</StyledTableRow>

</StyledTable>
