## Description

The `saveDiagram` operation saves the edits on a schematic diagram. It is performed on a [Schematic Diagram Template](/schematic-diagram-template) resource. This operation works from a [Schematic Diagram Content Object](/schematic-diagram-content-object) and returns a [Schematic Diagram Information Object](/schematic-diagram-information-object).

<Note type={"note"}>

Schematics services published without the Editing operation enabled don't support the `saveDiagram` operation.

</Note>

You can provide arguments to the `saveDiagram` operation as query parameters defined in the parameters table below.
