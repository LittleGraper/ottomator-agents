## Description

The `intersect` operation is performed on a [geometry service resource](/geometry-service). This operation constructs the set-theoretic intersection between an array of geometries and another geometry. The dimension of each resultant geometry is the minimum dimension of the input geometry in the `geometries` array and the other geometry specified by the `geometry` parameter. This operation calls `simplify` on the input `geometries` and `geometry`. You can provide arguments to the `intersect` operation as query parameters defined in the following parameters table.
