## Description

A subnetwork is updated by calling the `updateSubnetwork` operation. With this operation, one or all of the subnetworks in a single tier can be updated. When a subnetwork is updated, a trace is performed from all subnetwork controllers, the Subnetwork Name attribute is updated for all features in the subnetwork; the record representing the subnetwork inside the [SubnetLine class](https://pro.arcgis.com/en/pro-app/help/data/utility-network/subnetline-feature-class.htm) is refreshed; the [Subnetworks table](https://pro.arcgis.com/en/pro-app/help/data/utility-network/view-subnetworks-table.HTM) is updated; and [diagrams](https://pro.arcgis.com/en/pro-app/help/data/utility-network/about-network-diagrams.htm) are generated or updated for the subnetwork.  When a subnetwork is succcessfully updated it is considered clean (or consistent) and its record in the SubnetLine feature class, subnetwork system diagrams, and various fields for modified network features are updated and are in sync with changes made to the network topology. 

To learn more, see [Update a subnetwork](https://pro.arcgis.com/en/pro-app/latest/help/data/utility-network/update-a-subnetwork.htm).

`tierName`

(Optional)

`allSubnetworksInTier` 

(Optional)  

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Boolean parameter that is set to true when all the subnetworks in a tier need to be updated.  The default is false.

```
allSubnetworksInTier=true | false
```

(Optional)  

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Boolean parameter to continue updating subnetworks when `allSubnetworksInTier` is true and a failure occurs when processing a subnetwork.  The default is false.

```

```

## JSON Response syntax

Response when `async = false`:

```other
{
  "moment" : <datetime>,
  "failures" : [                // only if continueOnFailure = true
    {
      "subnetworkName" : <name>,
      "extendedError" : <long>,
      "message" : <error message>
    }
  ],
  "success" : <true | false>,
  "error" : {                   // only if success is false
    "extendedCode" : <HRESULT>,
    "message" : <error message>,
    "details" : [ <detail> ]
  }
}

```

 

Response when `async = true` :

```other
{
  "statusUrl" : <url>  
}

```

 

Response to the `statusURL` when either pending or in progress:

```other
{
  "status" : “<Pending | InProgress>”,
  "submissionTime" : <datetime>,
  "lastUpdatedTime" : <datetime>
}

```

 

Response to the status URL when completed:

```other
{
  "moment” : <datetime>,
  "status” : “Completed”,
  "submissionTime” : <datetime>,
  "lastUpdatedTime” : <datetime>,
  "failures" : [                // Only if continueOnFailure = true
    {
      "subnetworkName" : <name>,
      "extendedCode" : <HRESULT>,
      "message" : <error message>
    }
  ],
  "success" : <true | false>,
  "error" : {                   // only if success is false
    "extendedCode" : <HRESULT>,
    "message" : <error message>,
    "details" : [ <detail> ]
  }
}

```
