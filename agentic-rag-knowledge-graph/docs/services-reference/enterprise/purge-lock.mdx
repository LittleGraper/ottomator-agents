## Description

The `purgeLock` operation deletes all locks for the specified version.

<Note type={"caution"}>

If there are active edit sessions for the version, all unsaved edits will be lost. Review the [`version` ](/version) resource to determine if the version is being edited before using `purgeLock` .

</Note>

The following user accounts can run the `purgeLock` operation: 

-   Version owner
-   Service owner
-   Portal administrator

<Note type={"note"}>

Organization members must be assigned a license for the [ArcGIS Advanced Editing user type extension](https://enterprise.arcgis.com/en/portal/latest/administer/windows/license-user-type-extensions.htm#ESRI_SECTION1_B75DA80066744473AD5DD1312734654A) to use this operation for ArcGIS Enterprise 11.2 and later.

</Note>
