## Description

The `getImageUrl` operation returns an image URL that is accessible to the requesting web client. This operation is normally used after the [Find](/find-image-service) operation, which returns an image information list contains URL values. 

##  Request parameters

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`uri` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Specifies the URI of the image.

Example

```javascript
uri=/vsis3/ortho-maker/BrewertonTank/BrewertonTank/DJI_0707.JPG
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is `html` . 

Values: `html` \| `json` \| `pjson` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

The following is a sample request URL for the `getImageURL` operation:

```javascript
https://machine.domain.com/webadaptor/rest/services/BrewertonTank_cloud/ImageServer/getImageUrl?uri=/vsis3/ortho-maker/BrewertonTank/BrewertonTank/DJI_0707.JPG&f=html
```

 

## JSON Response syntax

```json
{
  "imageURL": <url>
}
```
