## Description

The `diagrams` resource represents all the schematic diagrams under a schematic service. It is returned as an array of [Schematic Diagram](/schematic-diagram) resources by the REST API. 

<Note type={"tip"}>

When there is a large number of schematic diagrams under the published schematic service, use the [searchDiagrams](/schematic-search-diagrams) operation to get a reduced list of schematic diagrams by filtering them out via a where clause by schematic folder, diagram template, and so on.

</Note>

## Example usage

Example 1: The schematic diagrams under the S1_Schematics public sample Schematics service

```javascript
https://myserver.esri.com/arcgis/rest/services/S1_Schematics/MapServer/exts/SchematicsServer/diagrams
```

 

Example 2: The schematic diagrams under the S2_InternalPlants public sample Schematics service

```javascript
https://myserver.esri.com/arcgis/rest/services/S2_InternalPlants/MapServer/exts/SchematicsServer/diagrams
```
