## Description

<Note type={"note"}>

<GUID_01CB5051_FA79_42A9_8275_EF008F2C3097_GUID_499B73EC_3D4E_46D2_863A_4534FAF79AA3 />

</Note>

The `namedTypes`  resource allows you to access and edit the entity types and relationship types defined as part of a knowledge graph service's [data model resource](/kgs-datamodel). You can add new entity types or relationship types and change the properties that can be specified for them if you have sufficient privileges to do so.

Use the data model [Query](/kgs-datamodel-query) operation to learn about the item types that are defined in the schema. Use the data model [Named Types](/kgs-datamodel-edit-namedtypes) resource with the name of a specific entity type or relationship type to access information about and modify settings for that graph item.

## Example usage

The following is a sample request URL used to get information about the data model resource's named types for a knowledge graph service named `SupplyChain` :

```javascript
https://machine.domain.com/webadaptor/rest/services/Hosted/SupplyChain/KnowledgeGraphServer/dataModel/edit/namedTypes?f=html
```
