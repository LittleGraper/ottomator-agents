## Overview

The [createReplica](/create-replica) and [synchronizeReplica](/synchronize-replica) responses include a `responseType`  parameter. This parameter indicates the type of information the response contains. The following table lists the possible values. 

##  Response types 

<StyledTable headers={"esriReplicaResponseType values,Description"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`esriReplicaResponseTypeData` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response contains data. 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`esriReplicaResponseTypeInfo` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response does not contain data. 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`esriReplicaResponseTypeEdits` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response contains edits. 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`esriReplicaResponseTypeNoEdits` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response does not contain edits or edit results. The is no delta file returned from the server in this case. 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`esriReplicaResponseTypeReplacementData` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response contains replacement data. 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`esriReplicaResponseTypeEditsAndData` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response contains both edits and data.

</StyledTableCell>

</StyledTableRow>

</StyledTable>
