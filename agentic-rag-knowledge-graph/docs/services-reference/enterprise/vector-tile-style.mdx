## Description

This resource returns styles for vector tiles in [Mapbox GL Style specification](https://www.mapbox.com/mapbox-gl-style-spec/#) version 8. The response for this styles resource includes the `sprite`  and `glyphs`  properties, with a relative path to the [Vector Tile Sprite](/vector-tile-sprite) and [Vector Tile Font](/vector-tile-font) resources. It also includes the `version`  property, which represents the version of the style specification.

## Example usage

Note: This is an example URL only and is not an active link to an existing attachment.

`https://services.myserver.com/arcgis/rest/services/Hosted/World/VectorTileServer/resources/style` 

## JSON Response syntax

```javascript
{
  "version": <version>,
   "sprite": "../sprites/sprite",
   "glyphs": "../fonts/{fontstack}/{range}.pbf",
   "sources": {...},
   "layers": [...]
  }
```
