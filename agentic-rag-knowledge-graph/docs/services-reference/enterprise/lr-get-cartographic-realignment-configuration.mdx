## Description

This operation retrieves the cartographic realignment configuration from the LRS dataset for the signed in ArcGIS Enterprise portal user.

The cartographic realignment configuration is stored in the linear referencing system (LRS) dataset. Anonymous user is not supported.

If [setCartoRealignConfiguration](/lr-set-cartographic-realignment-configuration) hasn't been called before calling this operation, the default value returned is `proportionalsnap`.

Learn more about [cartographic realignment configuration in Pipeline Referencing](https://pro.arcgis.com/en/pro-app/latest/help/production/location-referencing-pipelines/set-cartographic-realignment-options.htm) or [cartographic realignment configuration in Roads and Highways](https://pro.arcgis.com/en/pro-app/latest/help/production/roads-highways/set-cartographic-realignment-options.htm).
