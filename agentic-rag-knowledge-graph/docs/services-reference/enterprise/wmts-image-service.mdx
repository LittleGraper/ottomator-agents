## Description

This is the root `WMTS` resource and is available for tiled image services only. This resource is available as a Services Directory web page only. WMTS clients should access the [ WMTS Capabilities](/wmts-capabilities-image-service) and [ WMTS Tile](/wmts-tile-image-service) child resources.
