## Description

The iteminfo `thumbnail`  resource returns the thumbnail associated with the item information card of the service.

This resource streams the thumbnail contents to the client.

## New in 10.6.1

For feature and map services published using ArcGIS Pro (2.2 or later) layer / table resource on services support the `thumbnail`  resource. Check the `hasMetadata`  flag on a layer / table resource to determine if that layer / table supports the `thumbnail`  resource.
