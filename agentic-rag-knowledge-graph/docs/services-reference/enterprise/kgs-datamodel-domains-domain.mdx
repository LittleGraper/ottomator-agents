## Description

<Note type={"note"}>

<GUID_01CB5051_FA79_42A9_8275_EF008F2C3097_GUID_499B73EC_3D4E_46D2_863A_4534FAF79AA3 />

</Note>

The `domain`  resource allows you to update or delete a specific domain defined in a knowledge graph service's [data model resource](/kgs-datamodel) if you have sufficient privileges.

Use the data model [query](/kgs-datamodel-query) operation to learn about the domains that are defined in the schema. Use the data model [Domains](/kgs-datamodel-domains) resource with the name of the specific domain you want to access.

## Example usage

The following is a sample request URL to get information about the `WarehouseType`  domain that is defined in the data model resource for a knowledge graph service named `SupplyChain` :

```javascript
https://machine.domain.com/webadaptor/rest/services/Hosted/SupplyChain/KnowledgeGraphServer/dataModel/domains/WarehouseType?f=html
```
