## Description

Subnetwork controllers correspond to devices that have the Subnetwork Controller network capability set.  The `enableSubnetworkController` operation adds the subnetwork controller capability to the terminal of a device or junction object.  A subnetwork controller specifies the origin (or destination) of a subnetwork and defines how resources flow through a network. Examples of subnetwork controllers include circuit breakers in electric networks or town border stations in gas networks.
