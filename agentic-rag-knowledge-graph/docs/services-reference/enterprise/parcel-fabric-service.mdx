## Description

The parcel fabric service (`ParcelFabricServer`) exposes parcel management and parcel editing capabilities. These capabilities include parcel history management, building parcels and parcel editing oerations.

![Operations available in the parcel fabric service](./images/GUID-7AEBFE26-CFCF-4016-9266-5D71AF2CAD9C-web.png)

## Example usage

The following is a sample request URL for `ParcelFabricServer`:

`https://machine.domain.com/server/rest/services/CountyFabric/ParcelFabricServer`

 "name": "Parcel Fabric Server",
