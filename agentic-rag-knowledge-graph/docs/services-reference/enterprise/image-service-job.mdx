## Description

The Image Service job resource represents a job submitted using the [Export Tiles (Image Service)](/export-tiles-image-service) or [Estimate Export Tile Size (Image Service)](/estimate-export-tile-size-image-service) operations. It provides basic information about the job such as the job ID, status, and messages. Additionally, if the job has successfully completed, it provides information about the result parameters as well as the input parameters.

All result values can be accessed via the [Image Service Result](/image-service-result). The JSON response specifies a relative URL to the result resource with a `paramUrl` field. Similarly, all input parameter values are accessed via the [Image Service Input](/image-service-input). The JSON response specifies a relative URL to the input resource with a `paramUrl` field as well.

```json

```
