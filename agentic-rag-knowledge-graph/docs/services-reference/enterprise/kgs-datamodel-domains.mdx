## Description

<Note type={"note"}>

<GUID_01CB5051_FA79_42A9_8275_EF008F2C3097_GUID_499B73EC_3D4E_46D2_863A_4534FAF79AA3 />

</Note>

The `domains`  resource is available for a knowledge graph service's [data model resource](/kgs-datamodel). You can modify the collection of domains associated with the data model to define the content that can be provided in the property of an entity or relationship.

##  Request parameters 

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The only supported response format is `html` .

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

The following is a sample request URL used to get information about the domains associated with the data model resource for a knowledge graph service named `SupplyChain` :

```javascript
https://machine.domain.com/webadaptor/rest/services/Hosted/SupplyChain/KnowledgeGraphServer/dataModel/domains?f=html
```
