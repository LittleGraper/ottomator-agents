## Description

Create a version off of DEFAULT. The version is associated with the specified feature service.

<Note type={"note"}>

Organization members must be assigned a license for the [ArcGIS Advanced Editing user type extension](https://enterprise.arcgis.com/en/portal/latest/administer/windows/license-user-type-extensions.htm#ESRI_SECTION1_B75DA80066744473AD5DD1312734654A) to use this operation for ArcGIS Enterprise 11.3 and later.

</Note>

##  Request parameters 

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Specifies the output format of the response. The default response format is `html`.

Values: `html | json` | `pjson`

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`versionName`
(Required)

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Specifies the name of the new version.

The version name cannot exceed 62 characters or include any of the following special characters:

-   Period (.)
-   Semicolon (;)
-   Single quotation mark (')
-   Double quotation mark (")

Syntax: 
```other
versionName = <versionName>
```
</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`description`
(Optional)

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Specifies the description of the new version.

Ensure the description does not exceed 64 characters.

Syntax: `description = description for version` 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`accessPermission`
(Optional)

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Specifies the access permissions of the new version. The default access permission is `private`.

Values: `private` | `public` | `protected` | `hidden` 

Example: `private` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

Create a named version using the `create` operation.

Request URL and parameters:

`https://myserver.esri.com/server/rest/services/LandUse/VersionManagementServer/create` 

```javascript
f=json
versionName=carolina
description=Version for new subdivision edits
accessPermission=private

```
