## Description

The `htmlPopup`  resource provides details about the HTML pop-up authored by the user using ArcGIS Pro. This resource is available when a `dynamicLayer`  resource's `htmlPopupType`  property is not `esriServerHTMLPopupTypeNone` .

##  Request parameters

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`layer` 

(Required)

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Defines a dynamic layer. Only dynamic map layers are supported.

<Note type={"tip"}>

Use `gdbVersion`  on a dynamic map layer definition to specify an alternate geodatabase version. 

</Note>

Syntax

```javascript
{
  "id": <layerOrTableId>, // optional
  "source": <layer source>, // only dynamic map layers are supported
}
```

 

Example

```javascript
{
  "id": 101,
  "source": {
    "type": "mapLayer",
    "mapLayerId": 0,
    "gdbVersion": "SDE.DynamicLayers"
  }
}
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is `html` .

Values: `html`  \| `json`  \| `pjson` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

The following is a sample request URL used to access the `htmlPopup`  resource, querying a specific dynamic map layer:

```javascript
https://machine.domain.com/webadaptor/rest/services/USA/MapServer/dynamicLayer/1/htmlPopup?layer={"id":101,"source":{"type":"mapLayer","mapLayerId":2}}&f=pjson
```

 

## JSON Response syntax

```other
{
  "htmlPopupType": "<esriServerHTMLPopupTypeNone | esriServerHTMLPopupTypeAsURL | esriServerHTMLPopupTypeAsHTMLText>",
  "content": "<htmlContent>"
}
```

 

## JSON Response example

```other
{
  "htmlPopupType": "esriServerHTMLPopupTypeAsHTMLText",
  "content": "A Sample HTML pop up." 
}
```
