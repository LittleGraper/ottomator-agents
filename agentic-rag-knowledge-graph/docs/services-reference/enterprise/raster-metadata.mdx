## Description

The `metadata` resource returns metadata of the image service or a raster catalog item. The output format is always XML. 

## Example usage

Example 1: Return metadata of an image service: 

[https://sampleserver6.arcgisonline.com/arcgis/rest/services/Toronto/ImageServer/info/metadata](https://sampleserver6.arcgisonline.com/arcgis/rest/services/Toronto/ImageServer/info/metadata)

## JSON Response syntax

```javascript
Returns an xml.
          
```
