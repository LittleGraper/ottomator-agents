## Description

The Map Service job resource represents a job submitted using the [Export Tiles (Map Service)](/export-tiles-map-service) or [Estimate Export Tile Size (Map Service)](/estimate-export-tile-size-map-service) operations or a Map Service resource for consuming the result from an asynchronous Geoprocessing Service. If a job represents the `exportTiles`  or `estimateExportTilesSize`  operation, you will see basic information about the job such as the job ID, status, and messages. Additionally, if the job has successfully completed, it provides information about the result parameters as well as the input parameters.

<Note type={"note"}>

For more information on the properties returned in the job response, see [GP Job](/gp-job)

</Note>

All result values can be accessed via the [Map Service Result](/map-service-result). The JSON response specifies a relative URL to the result resource with a `paramUrl`  field. Similarly, all input parameter values are accessed via the [Map Service Input](/map-service-input). The JSON response specifies a relative URL to the input resource with a `paramUrl`  field as well.

## Cancel job

You can also cancel the job by making a request of the following form:

```javascript
https://machine.domain.com/webadaptor/rest/services/World_Street_Map/MapServer/jobs/<jobId>/cancel?f=pjson
```

 

When the job is cancelled, the following response is returned:

```javascript
{
  "jobId": "ja34307c9308f4144aed630b264582df1",
  "jobStatus": "esriJobCancelling"
}
```
