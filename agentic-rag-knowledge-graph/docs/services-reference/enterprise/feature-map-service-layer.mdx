## Description

An individual feature resource represents a single feature in a layer in a map service. A feature resource has two child resources:

-   [ Attachment Infos](/attachment-infos-map-service-layer)—Returns information about attachments associated with this feature. This resource is available only if the layer has advertised that it has attachments.
-   [ HTML Popup](/html-popup-map-service-layer)—Returns information about the HTML pop-up associated with this feature.

## Example usage

The following is a sample request URL used to access a feature resource:

```javascript
https://machine.domain.com/webadaptor/rest/services/Hurricanes/NOAA_Tracks_1851_2007/MapServer/0/1
```

 

## JSON Response syntax

```other
{
  "feature": <feature>
}
```
