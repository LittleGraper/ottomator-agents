## Description

The `convexHull` operation is performed on a [geometry service resource](/geometry-service). It returns the convex hull of the input geometries. The input geometries can be points, multipoints, polylines, or polygons, but they all must be the same type. The convex hull is typically a polygon but can also be a polyline or point in degenerate cases.

You can provide arguments to the `convexHull` operation as query parameters defined in the following parameters table.
