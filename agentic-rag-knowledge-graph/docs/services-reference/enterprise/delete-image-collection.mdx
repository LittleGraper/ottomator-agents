## Description

![Delete Image Collection diagram](./images/GUID-90875C4A-4B86-4040-A4B9-E3318200A84E-web.png)

The `DeleteImageCollection`  task deletes the image collection image service, that is, the portal-hosted image layer item. It will not delete the source images that the image collection references.

<Note type={"note"}>

As of ArcGIS 10.5, you must license your ArcGIS Server as an ArcGIS Image Server to use this resource.

</Note>
