## Description

<Note type={"note"}>

<GUID_01CB5051_FA79_42A9_8275_EF008F2C3097_GUID_499B73EC_3D4E_46D2_863A_4534FAF79AA3 />

</Note>

The `Type`  resource allows you to access information and modify settings for a specific entity type or relationship type defined in a knowledge graph service's [data model resource](/kgs-datamodel). You can add properties to an existing entity type or relationship type by defining a new field. You can include a property in the search index for the knowledge graph service by indexing the appropriate field. You can also update or delete an existing entity type or relationship type if you have sufficient privileges.

Use the data model [Query](/kgs-datamodel-query) operation to learn about the item types that are defined in the schema. Use the data model [Named Types](/kgs-datamodel-edit-namedtypes) resource with the name of the specific entity type or relationship type you want to access.

## Example usage

The following is a sample request URL to get information about the Warehouse entity type. Warehouse is one of the named types defined in the data model resource for a knowledge graph service named `SupplyChain` :

```javascript
https://machine.domain.com/webadaptor/rest/services/Hosted/SupplyChain/KnowledgeGraphServer/dataModel/edit/namedTypes/Warehouse?f=html
```
