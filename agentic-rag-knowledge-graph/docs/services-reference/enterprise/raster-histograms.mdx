## Example usage

The following is a [sample request URL](https://sampleserver6.arcgisonline.com/arcgis/rest/services/Toronto/ImageServer/1/info/histograms) used to access the `histograms`  resource:

```javascript
https://sampleserver6.arcgisonline.com/arcgis/rest/services/Toronto/ImageServer/1/info/histograms
```

 

## JSON Response syntax

Refer to [Histogram object](/histogram-object) for the syntax of a histogram.

```json
{
  "histograms": [
    <histogram1>,
    <histogram2>,
    <histogram3>,
    ...
  ]
}
```
