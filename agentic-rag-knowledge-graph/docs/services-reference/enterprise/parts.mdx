## Description

This resource returns a list of parts associated with an item. If an item has been committed, the parts list will be empty. 

## Request parameters 

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is html.

Values:`html | json`

</StyledTableRow>

</StyledTable>

## Example usage

Example: Item Parts resource

```other
https://<serverName>.<domain>.com/arcgis/rest/services/911CallsHotspot/GPServer/uploads/icb0817bf-53c7-45d9-9487-3b87b6cba253/parts
```

## JSON Response syntax

```other
{"parts": ["<partId1>","<partId2>","<partId3>","<partIdN>"]}
          
```
