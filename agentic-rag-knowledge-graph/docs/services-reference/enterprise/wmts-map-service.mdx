## Description

The `WMTS`  root resource is available for tiled map services only. This resource is available as a Services Directory web page only. WMTS clients should access the [ WMTS Capabilities](/wmts-capabilities-map-service) and [ WMTS Tile](/wmts-tile-map-service) child resources.
