## Description

The `locations` resource provides access to an operation that allows you to query the [locatability](https://pro.arcgis.com/en/pro-app/latest/help/data/utility-network/nonspatial-objects.htm#ESRI_SECTION1_A91E0889241F4CD4B49FB188A230573D) of a provided set of objects and optionally synthesize geometry to be returned.
