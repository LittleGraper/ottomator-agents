## Description

An image service provides access to raster data through a web service. Multiple rasters can be served as one image service through mosaic dataset technology, dynamically processed and mosaicked on the fly. An image service supports accessing the mosaicked image, its catalog, and the individual rasters in the catalog. 

An image service supports dynamic access and tiled access. Dynamic access provides more functionality, and tiled access provides faster and more scalable access to precooked tiles. There are two types of tiles: map tiles and elevation tiles. Map tiles (introduced at 10.1) represent rendered map images, typically in JPEG or PNG format. Elevation tiles (introduced at 10.3) represent elevation data and are stored as Limited Error Raster Compression (LERC) tiles. 

The ArcGIS REST API `ImageServer`  resource root information represents an image service published with ArcGIS Server. The resource provides basic information associated with the image service, such as the service description, its name, its extent, pixel sizes, and band counts. 

## Child operations

The `ImageServer`  resource supports the following operations

-   [Add Rasters](/add-rasters)—Add new rasters to the image service.
-   [Compute Cache Info](/compute-cache-info)—Compute and generate new image tile cache schemes for the image service.
-   [Calculate Volume](/calculate-volume)—Compute a volumetric calculation on the elevation data.
-   [Compute Class Statistics](/compute-class-statistics)—Calculate sample classes’ pixel statistics of the image service for composing spectral signatures of defined classes
-   [Compute Histograms](/compute-histograms)—Calculate histograms of the image service at given geometry extent.
-   [Compute Multidimensional Info](/compute-multidimensional-info)—Construct a multidimensional info object for an image service of mosaic dataset based on its catalog table.
-   [Compute Pixel Location](/compute-pixel-location)—Calculate pixel location in columns and rows for an image based on input geometries.
-   [Compute Statistics and Histograms](/compute-statistics-and-histograms)—Calculate both statistics and histograms of the image service at given geometry extent.
-   [Compute Tie Points](/compute-tie-points)—Get tie points that can be used to match the source image to the reference image of the image service.
-   [Delete Rasters](/delete-rasters)—Delete one or more rasters from the image service.
-   [Download Rasters](/download-rasters)—Download source rasters in original or other formats with given geometry extent.
-   [Export Image](/export-image)—Dynamically generate an image based on client-specified band selection, format, mosaic rule, and rendering rule, etc.
-   [Export Tiles](/export-tiles-image-service)—Perform as an asynchronous task and allow client applications to download map tiles from the server for offline use.
-   [Find (Image Service)](/find-image-service)—Finds all the image services that contain toGeometry and sorts them accordingly.
-   [Get Samples](/get-samples)—Get sample point locations, pixel values, and corresponding spatial resolutions of the image service source data for a given geometry.
-   [Get Image Url](/get-image-url)—Get a list of urls after another image service operation such as Find (Image Service) is run.
-   [Identify](/identify-image-service)—Identify the content of an image service for given locations, given mosaic and rendering rules (for example, pixel values and raster visibilities).
-   [Image to Map](/image-to-map)—Converts a geometry on image space to map space.
-   [Image to Map Multiray](/image-to-map-multiray)—Computes as 3D geometry in a map from multiple image space geometries on multiple corresponding raster items of the same object.
-   [Map to Image](/map-to-image)—Converts a geometry on a map space to an image space.
-   [Measure](/measure)—Measure height, distance, direction, area, and perimeter of the image service.
-   [Measure from Image](/measure-from-image)—Provides mensuration capability within one image space and returns the measurement result in map space unit.
-   [Project](/project)—Project an array of input geometries from the input spatial reference to the output spatial reference.
-   [Query Boundary](/query-boundary)—Calculate the boundary geometry and its area of coverage of the image service.
-   [Query (GPS)](/query-gps)—Queries generic camera information, camera GPS locations when taking an image, image center coordinates, and exterior orientation information of each source image inside the image collection. 
-   [Query (Image Service)](/query-image-service)—Query an image service resource by applying a specified filter.
-   [Update Raster](/update-raster)—Update a raster in an image service. (for example, attributes, footprints, or existing raster files).
-   [Uploads](/uploads)—Upload raster files that can be used in add and update raster operations.
-   [Validate](/validate)—Validate rendering and mosaic rules using the image service.

## Child resources

The `ImageServer`  resource provides the following child resources.

-   [Colormap](/colormap)—Return the color map (RGB color representation of pixel values) attached to the image service or its raster function templates.
-   [Histograms](/histograms)—Return the histogram attached to the image service.
-   [Image Tile](/image-tile)—Return a precooked image tile at a specific level, row, and column combination.
-    Image Service [Info](/info)—Provide access to the following image service information resources: iteminfo, metadata, and thumbnail.

    -   [Item Information](/item-information)—Return the item information card for the service, such as description, summary, and tags.
    -    Image Service [Metadata](/metadata)—Return XML formatted image service metadata info.
    -    Image Service [Thumbnail](/thumbnail)—Return the thumbnail image of the image service.
-   [Key Properties](/key-properties)—Return the key properties of the image service.
-   [KML Image](/kml-image-image-service)—Return a ground overlay document wrapped in a `.kmz`  file.
-   [Legend](/legend-image-service)—Return the legend of the image service.
-   [Multidimensional Info](/multidimensional-info)—Return the variables and dimensions information of the image service.
-   [Raster Attribute Table](/raster-attribute-table)—Return an attribute table attached to the image service or its raster function templates.
-   [Raster Catalog Item](/raster-catalog-item)—Return catalog field values of the selected raster item.

    -   [Raster Image](/raster-image)—Return a composite image for the selected raster catalog item.
    -   [Raster Info](/raster-info)—Return basic raster information of the selected raster catalog item, such as its width, height, number of bands, and pixel type.

        -   [Image Support Data](/image-support-data)—Return the NITF file structure and contents in XML format to provide more detailed information about a particular NITF file.
        -   [Raster Thumbnail](/raster-thumbnail)—Return a thumbnail for the selected raster catalog item.
        -   [Raster Metadata](/raster-metadata)—Return metadata for the selected raster catalog item.
        -   [Raster Key Properties](/raster-key-properties)—Return key properties for the selected raster catalog item.
        -   [Raster Histograms](/raster-histograms)—Return histograms for the selected raster catalog item.
        -   [Raster ICS](/raster-ics)—Return the image coordinate system for the selected raster catalog item.
        -   [Raster ICS To Pixel](/raster-ics-to-pixel)—Return geometric transformation coefficients that can be used to perform image coordinate system \[sensor: Frame] to pixel (column, row) transformation for the selected raster catalog item.
-   [Raster Function Info](/raster-function-infos)—Return available raster function templates attached to the image service and its source mosaic dataset.
-   [Raster File](/raster-file)—Return the raw raster file resource of the image service.
-   [Slices](/slices)—Return the slice IDs and corresponding multidimensional definition information of the image service.
-   [Statistics](/statistics)—Return the statistics attached to the image service.
-   [WMTS](/wmts-tile-image-service)—Return the WMTS resource of the image service.
