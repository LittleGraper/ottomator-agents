## Description

This operation retrieves the calibration configuration from the LRS dataset for the signed in ArcGIS Enterprise portal user.

Event behaviors are applied for manual calibration point edits if calibration configuration is set to `include`. Anonymous user is not supported.

If [setCalibrationConfiguration](/lr-set-calibration-configuration) hasn't been called before calling this operation, the default value returned is `include`.

Learn more about [excluding manual calibration from event behaviors in Pipeline Referencing](https://pro.arcgis.com/en/pro-app/latest/help/production/location-referencing-pipelines/add-calibration-points.htm#GUID-84557E86-2F7F-431E-83E9-4F2F0A76E1CA) or about [excluding manual calibration from event behaviors in Roads and Highways](https://pro.arcgis.com/en/pro-app/latest/help/production/roads-highways/add-calibration-points.htm#GUID-84557E86-2F7F-431E-83E9-4F2F0A76E1CA).
