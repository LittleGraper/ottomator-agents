## Description

The `NetworkDiagramServer` service exposes the network diagram capabilities to be consumed by internal and external web applications. This resource represents a network diagram service published with ArcGIS Server. It provides information about the service itself (name, type, default diagram template) and exposes various functions to access published network diagrams, create new network diagrams and store them, edit and maintain network diagrams, and so on.

This resource supports some operations which allow retrieving network diagrams, getting the characteristics of the diagrams you want (diagram info, consistency state), creating new network diagrams and deleting network diagrams:

-   [Find Diagram Names](/find-diagram-names-network-diagram-service)—Retrieves any network diagrams stored in the database that cover a given extent, verify an SQL where clause, and/or include a set of particular network element Global IDs.
-   [Find Diagram Infos](/find-diagram-infos-network-diagram-service)—Returns the set of [Diagram Info](/appendix-diagram-information-object-network-diagram-service) objects for a diagram list.
-   [Query Consistency State](/query-consistency-state-network-diagrams)—Returns a set of consistency states for a diagram list.
-   [Create Diagram From Features](/create-diagram-from-features-network-diagram-service)—Creates a new network diagram from a set of network element Global IDs.
-   [Delete Diagram](/delete-diagram-network-diagram-service)—Deletes a network diagram.

The Network Diagram service supports multiple child resources:

-   [Network Diagrams](/network-diagrams)—Provides access to the network diagrams published with ArcGIS Server
-   [Diagram Templates](/diagram-templates-network-diagram-service)—Provides access to the diagram templates
-   [Diagram Dataset](/diagram-dataset-network-diagram-service)—Regroups the info related to each diagram template under a Network Diagram service

<Note type={"note"}>The active portal account must be licensed with the [ArcGIS Advanced Editing](https://enterprise.arcgis.com/en/portal/latest/administer/windows/license-user-type-extensions.htm#ESRI_SECTION1_B75DA80066744473AD5DD1312734654A) user type extension to use network diagrams with the utility network or trace network services.</Note>
