#### Object properties

<StyledTable headers={"Property,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `name`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The diagram name (String).

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `id`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The diagram identifier in the schematic service (String).

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `templateIdentifier`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The identifier of the diagram template on which the schematic diagram is based in the schematic service (String).

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `templateObjectID`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The OBJECTID of the diagram template on which the schematic diagram is based in the schematic dataset (Long).

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `schematicLayerIdentifier`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The identifier of the schematic diagram layer in the schematic service (Long).

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `folderObjectID`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The OBJECTID of the schematic folder where the diagram is stored in the schematic dataset (Long).

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `folderIdentifier`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The identifier of the schematic folder where the diagram is stored in the schematic service (String).

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `objectID`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The OBJECTID of the schematic diagram in the schematic dataset (Long).

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `createdBy`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the diagram creator (String).
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `lastModificationBy`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the last user who modified the diagram (String).
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `lastUpdateBy`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the last user who updated the diagram (String).
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `creationDate`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The diagram creation date (String).
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `lastModificationDate`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The date the diagram was last modified (String).
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `lastUpdateDate`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The date the diagram was last updated (String).
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `lockedBy`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the last user who locked the diagram (String).
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `version`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The SDE version on which the diagram is based (String).
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `extent`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The diagram extent ([envelope](/geometry-objects)).
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `rootNodes`

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The list of the schematic root node identifiers (Array of String).
      
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `endNodes`

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The list of the schematic end node identifiers (Array of String).
      
    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
