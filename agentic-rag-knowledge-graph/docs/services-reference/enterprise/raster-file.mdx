## Description

The Raster File resource represents a single raw raster file. The `id`  required to request the file can be obtained using the [Download Rasters operation](/download-rasters).

This resource streams the file contents to the client.

##  Request parameters 

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`id` 

(Required)

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Description: The ID of the raster file. This ID is obtained using the [ Download Rasters operation](/download-rasters).

Example: Suppose the Download Rasters operation returned the following response: 

```javascript
{
  "rasterFiles": [
   {
    "id": "t1923.png",
    "rasterIds": [54],
    "size": 65417
   },
   {
    "id": "t1923.pgw",
    "rasterIds": [54],
    "size": 765368
   },
   {
    "id": "t1923.png.aux.xml",
    "rasterIds": [54],
    "size": 853
   },
   {
    "id": "t1928",
    "rasterIds": [60],
    "size": 786296
   }
  ]
}
```

 

You can download the second file in this response by specifying its `id`  as the query parameter for this resource: `id=t1923.pgw` .

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`rasterId` 

(Required)

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Description: The object ID of the raster item. This ID is obtained using the [ Download Rasters operation](/download-rasters). This is required if the source raster is a GRID, WCS, or geodatabase raster.

Example: You can download the fourth raster from the example response by specifying its `id` and `rasterId`  as the query parameters for this resource: `id=t1928&rasterId=60`.

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

Example 1: Access the raster file resource for ID: https&#x3A;//myserver.mydomain.com/1n2w13w.jpg. 

`https://myserver.mydomain.com/ArcGIS/rest/services/ImageServiceName/ImageServer/file?id=https://servername/1n2w13w.jpg&rasterId=61` 

Example 2: Access a geodatabase raster resource.

`https://myserver.mydomain.com/ArcGIS/rest/services/ImageServiceName/ImageServer/file?id=t1928&rasterId=60`
