## Description

![Uninstall Deep Learning Model](./images/GUID-0D76CC8A-8C20-41A5-A69A-923A436BA44A-web.png)

The `UninstallDeepLearningModel`  operation is used to uninstall the uploaded deep learning model package (`.dlpk` ) from the raster analysis image server. It will delete the named deep learning model from the image server's configuration store but not the portal item.

## Example usage

Below is a sample request URL for `UninstallDeepLearningModel` .

```javascript
https://services.myserver.com/arcgis/rest/services/System/RasterAnalysisTools/GPServer/UninstallDeepLearningModel/submitJob
```

 

## JSON Response example

The response returns the `installSucceed`  output parameter, which has properties for parameter name, data type, and value. The content of the value is the portal item ID of the uninstalled model.

```javascript
{
 "paramName": "installSucceed",
 "dataType": "GPString",
 "value": "f0e602cf29dd4674ad814773ed361fb6"
}
```
