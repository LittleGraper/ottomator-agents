## Description

The `replicas`  resource lists all replicas in the geodatabase that have been created via [sync-enabled](/using-sync) feature services. This list includes the replica name and replica ID of each replica. The list of replicas is only shown if logged in as an admin user. Otherwise, the list is empty.

```markdown
https://services.myserver.com/ERmEceOGq5cHrItq/ArcGIS/rest/services/USA/GeoDataServer/replicas?f=pjson
```
