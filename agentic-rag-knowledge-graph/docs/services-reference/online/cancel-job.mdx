## Description

The `cancel`  job operation supports cancelling a job while `updateTiles`  is running from a hosted map service. The result of this operation is a response indicating success or failure with error code and description.

## JSON Response syntax

```javascript
{
  "success": <true|false>,
  "error": {
    "code": <code>,
    "message": "<description>",
    "details": [
      "<message>"
    ]
  }
}
          
```

 

## JSON Response example (when `cancel`  job succeeds)

```javascript
{
  "success" : true
}
          
```

 

## JSON Response example (when `cancel`  job fails)

```javascript
{
  "error" : 
  {
    "code" : 400, 
    "message" : "", 
    "details" : [
      "Unable to cancel job."
    ]
  }
}
          
```
