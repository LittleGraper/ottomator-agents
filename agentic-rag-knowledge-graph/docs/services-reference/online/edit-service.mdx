## Description

The `edit`  operation enables editing the service `sourceItemID` , `minScale` , `maxScale` , `exportTilesAllowed` , and `maxExportTilesCount`  parameters.

##  Parameters 

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">`f`</StyledTableCell>
    <StyledTableCell cellStyle="markdown">
      Description: The response format. The default response format is `html` .

Values: `html`  \| `json`  \| `pjson` 
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">`serviceDefinition`</StyledTableCell>
    <StyledTableCell cellStyle="markdown">
      Description: The service definition.

      {'{'}

      "exportTilesAllowed": false,

      "maxExportTilesCount": 100000,

      "cacheOnDemand": false,

      "capabilities": "Map",

      "cacheMaxAge": 600,

      "abstract": null,

      "metadataLink": null,

      "keywords": \[]

      {'}'}
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">`sourceItemID`</StyledTableCell>
    <StyledTableCell cellStyle="markdown">
      Description: The source item ID is the GeoWarehouse item ID of the map service.

      `(Source Item ID) 087f5b67b1024523ae91f1069f294364` 
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">`minScale`</StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      `(MinScale) 591657527.591555` 
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">`maxScale`</StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      `(MaxScales) 1128.497176` 
    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">`exportTilesAllowed`</StyledTableCell>
    <StyledTableCell cellStyle="markdown">
      Description: Enables or disables the exporting of tiles for a hosted map service.

      `{exportTilesAllowed : true | false}`

  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">`maxExportTilesCount`</StyledTableCell>

      `{maxExportTilesCount : 500000}`

  </StyledTableRow>

## Example usage

<Note type={"note"}>

This is an example URL only and is not an active link to an existing service.

</Note>

`https://services.myserver.com/arcgis/rest/admin/services/serviceName/MapServer/edit?f=html` 

## Example 1

<Note type={"note"}>

This is an example of how to edit the `serviceDefinition`  "maxExportTilesCount" and "exportTilesAllowed".

</Note>

`https://services.myserver.com/arcgis/rest/admin/services/serviceName/MapServer/edit?serviceDefinition={"maxExportTilesCount":5000,"exportTilesAllowed":true}&token=AddYourToken`
