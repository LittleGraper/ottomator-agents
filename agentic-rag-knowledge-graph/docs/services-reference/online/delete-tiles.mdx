## Description

The `deleteTiles`  operation supports deleting tiles from an existing cached service.

## JSON Response Syntax

```javascript
{
  "id":<id>,
  "name": <service name>,
  "itemId": <item id>,
  "status": <success | failed>,
  "type": <map service>,
  "jobId": <job id>,
  "message" : "<message>"
}
or 
{
  "error": {
    "code": <code>,
    "message": "<description>",
    "details": [
      "<message>"
    ]
  }
}
```

{
  "id" : "82cc6785db3dcd0d",

## JSON Response Example (fails)

```javascript
{
  "error" : 
  {
    "code" : 404, 
    "message" : "Unable to delete tiles.",
    "details" : [ "'levels' parameter cannot be empty" ]
  }
}
          
```
