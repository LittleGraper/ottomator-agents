<ParamMetadata types={[{name:"File",link:"../routing-data-types#file"}]} />

Provides access to a `.zip` file that contains a file geodatabase containing the inputs and outputs of the analysis in a format that can be used to share route layers with ArcGIS Online or Portal for ArcGIS. The parameter value is populated only when the [`save_route_data`](#save_route_data) parameter is set to `true`.
