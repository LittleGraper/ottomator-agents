Specifies the output turn-by-turn directions for the routes calculated in the analysis, represented as point locations along the routes where specific direction events or maneuvers occur.

<details>

 <If c="requestType == 'job'">
 <summary><span class="font-sm text-color-link">Show attributes for `output_direction_points` output parameter</span></summary>
 </If>
 
 <If c="requestType == 'direct'">
 <summary><span class="font-sm text-color-link">Show attributes for `directionPoints` output parameter</span></summary>
 </If>

-   <p class="space-x-2"><span class="font-bold">RouteID</span><span class="font-sm text-color-3">integer</span><span class="font-bold text-[#00619b]">nullable</span></p>

    <If c="requestType == 'job'">
    The `ObjectID` of the output `output_routes` feature with which this direction point is associated.
    </If>

    <If c="requestType == 'direct'">
    The `ObjectID` of the output `routes` feature with which this direction point is associated.
    </If>

-   <p class="space-x-2"><span class="font-bold">Sequence</span><span class="font-sm text-color-3">integer</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The sequence of the direction points for the route, starting with 1.

-   <p class="space-x-2"><span class="font-bold">DirectionPointType</span><span class="font-sm text-color-3">integer</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The type of direction event or maneuver described by the point, designated by one of the values below:

    <details>

    <summary><span class="font-sm text-color-link">Show values</span></summary>
    -   Header (1)
    -   Arrive (50)
    -   Depart (51)
    -   Straight (52)
    -   On Ferry (100)
    -   Off Ferry (101)
    -   Central Fork (102)
    -   Roundabout (103)
    -   U-Turn (104)
    -   Door (150)
    -   Stairs (151)
    -   Elevator (152)
    -   Escalator (153)
    -   Pedestrian Ramp (154)
    -   Left Fork (200)
    -   Left Ramp (201)
    -   Clockwise Roundabout (202)
    -   Left-handed U-turn (203)
    -   Bear left (204)
    -   Left Turn (205)
    -   Sharp Left (206)
    -   Left Turn and immediate Left Turn (207)
    -   Left Turn and immediate Right Turn (208)
    -   Right Fork (300)
    -   Right Ramp (301)
    -   Counter-Clockwise Roundabout (302)
    -   Right-handed U-turn (303)
    -   Bear right (304)
    -   Right Turn (305)
    -   Sharp Right (306)
    -   Right Turn and immediate Left Turn (307)
    -   Right Turn and immediate Right Turn (308)
    -   Up Elevator (400)
    -   Up Escalator (401)
    -   Up Stairs (402)
    -   Down Elevator (500)
    -   Down Escalator (501)
    -   Down stairs (502)
    -   General Event (1000)
    -   Landmark (1001)
    -   Time Zone Change (1002)
    -   Traffic Event (1003)
    -   Scaled Cost Barrier Event (1004)
    -   Boundary Crossing (1005)
    -   Restriction Violation (1006)
    -   Lane (1007)
    </details>

-   <p class="space-x-2"><span class="font-bold">StopID</span><span class="font-sm text-color-3">integer</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The `ObjectID` of the stop with which this direction point is associated, if any. If the point does not represent a visit to a stop, the value is null.

-   <p class="space-x-2"><span class="font-bold">DisplayText</span><span class="font-sm text-color-3">string (length: 1024)</span><span class="font-bold text-[#00619b]">nullable</span></p>    

    The directions text to display in the consuming application.

<If c="requestType=='job'">
-   <p class="space-x-2"><span class="font-bold">ArrivalTime</span><a class="font-sm text-color-3" href="../routing-data-types#datetime">datetime</a><span class="font-bold text-[#00619b]">nullable</span></p>

    The time the direction event occurs in coordinated universal time (UTC).
</If>

<If c="requestType=='direct'">
-   <p class="space-x-2"><span class="font-bold">ArrivalTime</span><a class="font-sm text-color-3" href="../routing-data-types#datetime">datetime</a><span class="font-bold text-[#00619b]">nullable</span></p>

    The time the direction event occurs in coordinated universal time (UTC).
</If>

-   <p class="space-x-2"><span class="font-bold">ArrivalUTCOffset</span><span class="font-sm text-color-3">number (non-negative)</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The difference in minutes between the local time at the maneuver location and UTC time shown in the `ArrivalTime` field.

-   <p class="space-x-2"><span class="font-bold">Name</span><span class="font-sm text-color-3">string (length: 1024)</span><span class="font-bold text-[#00619b]">nullable</span></p> 

    The name of the direction point.

-   <p class="space-x-2"><span class="font-bold">ExitName</span><span class="font-sm text-color-3">string</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The highway exit name that appears in the directions instruction.

-   <p class="space-x-2"><span class="font-bold">AlternateName</span><span class="font-sm text-color-3">string</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The alternate source name that appears in the directions instruction.

-   <p class="space-x-2"><span class="font-bold">IntersectingName</span><span class="font-sm text-color-3">string</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The name of the intersecting or cross street that appears in the directions instruction.
    
-   <p class="space-x-2"><span class="font-bold">BranchName</span><span class="font-sm text-color-3">string</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The signpost branch name that appears in the directions instruction.

-   <p class="space-x-2"><span class="font-bold">TowardName</span><span class="font-sm text-color-3">string</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The signpost toward destination name that appears in the directions instruction.

-   <p class="space-x-2"><span class="font-bold">Level</span><span class="font-sm text-color-3">integer</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The building level at which this direction event occurs. This value corresponds to the `Level` property defined in the network dataset used for the analysis.

-   <p class="space-x-2"><span class="font-bold">ShortVoiceInstruction</span><span class="font-sm text-color-3">string (length: 1024)</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The short text to use as voice guidance text in the consuming application.

-   <p class="space-x-2"><span class="font-bold">VoiceInstruction</span><span class="font-sm text-color-3">string (length: 1024)</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The additional text, including expanded abbreviations and plurals, to use as voice guidance text in the consuming application.

-   <p class="space-x-2"><span class="font-bold">Azimuth</span><span class="font-sm text-color-3">number (non-negative)</span><span class="font-bold text-[#00619b]">nullable</span></p>

    The bearing in degrees of the vehicle departing this point. Zero indicates north.

</details>
