{/* <p class="space-x-2"><span class="font-sm text-color-3">type:bool</span></p> */}

Specify whether the `TimeWindowStart` and `TimeWindowEnd` attribute values on [`stops`](#stops) are specified in UTC or geographically local time. 

See the [startTimeIsUTC](#starttimeisutc) parameter for an explanation of UTC versus local time.

<If c="publication=='services-reference/enterprise'">
<Note type={"note"}>

This parameter is ignored when impedanceAttributeName is set to distance units.

</Note>
</If>

<Note type={"note"}>

The service does not allow the combination of `timeWindowsAreUTC=true`  and `startTimeIsUTC=false` . An error is produced if this combination is used. 

</Note>
