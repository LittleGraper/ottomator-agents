**Allowed values:** `Meters`, `Kilometers`, `Feet`, `Yards`, `NauticalMiles`, `Seconds`, `Minutes`, `Hours`, `Days`

{/* for Routes */}
<If c={props.route}>

Specify the units that should be used to report the total travel time or travel distance for the output routes.

The service determines the shortest path based on travel time or travel distance along the streets depending on whether the units you specify for this parameter are time- or distance-based.

<Note type={"note"}>
The units of the attribute value of `AdditionalTime` and `AdditionalDistance` on the [stops](#stops) parameter are same as the units of the `measurement_units` .
</Note>

</If>

{/* for closest facility */}
<If c={props.closestFacility}>

Specify the units that should be used to report the total travel time or travel distance for the output routes.

The service finds the closest facility based on the travel time or the travel distance along the streets, depending on whether the units you specify as the value for this parameter are time- or distance-based.

<Note type={"note"}>
The units of [cutoff](#cutoff)  parameter and the attribute values of `AdditionalTime`, `AdditionalDistance`, and `Cutoff` on the [incidents](#incidents)  and [facilities](#facilities)  parameter are same as the units of the `measurement_units`.
</Note>
</If>

{/* for location allocation */}
<If c={props.locationAllocation}>

Specify the units that will be used to measure the travel times or travel distances between demand points and facilities. 

The service finds the best facilities based on those that can reach, or be reached by, the most amount of weighted demand with the least amount travel. The output allocation lines report travel distance or travel time in different units, including the units you specify for this parameter.

<Note type={"note"}>
The units of the [`default_measurement_cutoff`](#default_measurement_cutoff)  parameter and the attribute values of `Cutoff` on the [`facilities`](#facilities) and [`demand_points`](#demand_points)  parameter are same as the units of the `measurement_units`.
</Note>

</If>
