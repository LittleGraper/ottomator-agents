The use of all relational data store operations in the ArcGIS Server Admin API has been deprecated. Starting at Insights 2021.1, the registration and management of relational data store types should be performed with ArcGIS Insights. For more information, see [Manage connector types](https://doc.arcgis.com/en/insights/latest/administer/manage-connector-types.htm).
