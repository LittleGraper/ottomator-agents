<Note type="legacy">

    Big data file shares (BDFS) and the ArcGIS GeoAnalytics Server extension are
    [deprecated](https://support.esri.com/en-us/knowledge-base/deprecation-arcgis-geoanalytics-server-000032771) at
    ArcGIS Enterprise 11.4. The last release to support both BDFS and GeoAnalytics server is ArcGIS Enterprise 11.3.

</Note>
