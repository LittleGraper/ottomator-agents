## Description

The `unregisterItem` operation unregisters a [data item](/dataitem) that has been previously registered with the server's data store.

<Note type="caution">

At ArcGIS Enterprise 11.4, an object store is considered to be part of an ArcGIS Enterprise base deployment. An object store is required to publish hosted scene layers and hosted 3D tile layers. Attempting to unregister an object store that scene or 3D tile services reference will result in an error message, which lists the services that will either need to be deleted or have their [web layers replaced](https://pro.arcgis.com/en/pro-app/latest/tool-reference/server/replace-web-layer.htm) to reference a different object store. An object store cannot be uregistered until there are no longer any services using it.

</Note>

##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `itempath` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The item path for the data item.

      ```other label=Example
      itempath=/fileShares/folder_share
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `force` 

      (Required)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      To unregister a data store item, this parameter must be set to `true`.

      Value: `true` 

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`. 

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `unregisterItem` operation:

```other
POST /webadaptor/admin/data/unregisterItem HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

itempath=/fileShares/folder_share&force=true&f=pjson
```
