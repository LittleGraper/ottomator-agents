## Description

The `system` resource is a collection of miscellaneous server-wide resources such as [Server Properties](/serverproperties), [ Server Directories](/directories), the [Configuration Store](/configstore), [Web Adaptors](/webadaptors), and [Licenses](/licenses) resources.

  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL for the `system` resource:

```markdown
https://machine.domain.com/webadaptor/admin/system?f=json
```
