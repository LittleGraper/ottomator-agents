## Description

<Note type={"note"}>

The Message Bus service was removed at 10.7.1. This operation is only functional in versions 10.5 through 10.7.

</Note>

The health check returns information for each machine that participates in the Message Bus service. While this operation is exposed, it will not return any valuable information.
