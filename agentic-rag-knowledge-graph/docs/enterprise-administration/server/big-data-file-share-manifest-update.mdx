## Description

<GAX_DEPRECATION_NOTICE />

The `update` operation uploads a manifest for a big data file share item that will replace the existing manifest for the big data file share item.

##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `manifest` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The manifest file to be uploaded. 

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `fileData` 

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The JSON representing the updated manifest. This can be used in place of the manifest file.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`. 

      Values: `html` | `json` | `pjson` 

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
