## Description

The `config` resource returns the shared key that is used by all the Web Adaptors to encrypt key data bits in the incoming requests to the server. The shared key can be updated using the [Update Web Adaptors Configuration](/updatewebadaptorsconfig) operation.

    <StyledTableCell cellStyle="markdown">

## Example usage

The following is a sample request URL used to access the `config` resource:

```markdown
https://machine.domain.com/webadaptor/admin/system/webadaptors/config?f=pjson
```
