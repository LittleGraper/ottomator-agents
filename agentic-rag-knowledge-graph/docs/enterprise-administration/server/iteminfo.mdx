## Description

The `iteminfo` resource stores metadata about a [service](/service). Typically, this information is available to clients that want to index or harvest information about the service. Item information is represented in JSON format defined by the schema described below. It can optionally contain thumbnail images associated with the service. The thumbnail images can be accessed with a URL by appending the path of the image to this item information URL. 

If the thumbnail is stored under `thumbnail/image.jpg`, the URL to access this image will be as follows:

```other
https://machine.domain.com/webadaptor/admin/services/<folder>/<service>/iteminfo/thumbnail/image.jpg
```

## Example usage

The following is a sample request URL used to access the `itemInfo` resource:

```markdown
https://machine.domain.com/webadaptor/admin/services/Maps/Test.MapServer/itemInfo?f=pjson
```

  "summary": "<summary>",
  "title": "<title>",
  "tags": "<tags>",

    "<keyword1>",
