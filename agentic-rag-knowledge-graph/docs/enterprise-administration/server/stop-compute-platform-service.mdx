## Description

<GAX_DEPRECATION_NOTICE />

The `stop` operation stops your compute platform service.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `stop` operation:

```markdown
POST /webadaptor/admin/system/platformservices/f44d878f-1341-4444-aa1d-53597f6756c3/stop HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
