## Description

The resource for an individual job returns the asynchronous execution of an operation by a geoprocessing service. From this resource, you can [cancel the job](/cancel-job) if its current status is `SUBMITTED` or `EXECUTING` (keeping the job information in the system), or [delete the job](/delete-job) regardless of its current status, which will remove all trace of it from the service and cancel the job if applicable.

## Response properties

<StyledTable headers={"Property,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `jobID`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The unique ID of the job, which is used for its URL.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `submittedTime` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The time the job was submitted in milliseconds from epoch.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `endTime` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The time the job was completed, either as a failure or a success.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `status` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The current status of the job.

      Values: `esriJobNew` | `esriJobSubmitted` | `esriJobExecuting` | `esriJobSucceeded` | `esriJobFailed` | `esriJobCancelling` | `esriJobCancelled` | `esriJobWaiting`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `username`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The ArcGIS Server user who submitted the job. If the service is anonymous, this value is `null`.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `machinename` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The machine running the job.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `url` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The URL of the job's page in the ArcGIS Server Services Directory. This page displays all messages logged during the job execution.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access an individual job resource:

```markdown
https://machine.domain.com/webadaptor/admin/services/Maps/Test.MapServer/jobs/j290c2d4e60364xx2a0c735fb9b68428a?f=pjson
```
