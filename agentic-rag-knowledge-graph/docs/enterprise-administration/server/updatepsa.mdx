## Description

The `update` operation changes the account properties of the primary site administrator account. 

<Note type={"note"}>

The primary site administrator account is stored within ArcGIS Server and can be used to log into ArcGIS Server Manager and perform administrative workflows. This operation must be invoked over an SSL connection as it requires an exchange of vital credentials.

</Note>


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `username` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      You can optionally provide a new name for the primary site administrator account.

      ```markdown label=Example
      username=agsadmin
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `password` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The password for the new primary site administrator account.

      ```markdown label=Example
      password=newSecret
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`. 

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `update` operation:

```markdown
POST /webadaptor/admin/security/psa/update HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

username=agsadmin&password=newSecret&f=pjson
```
