## Description

<GAX_DEPRECATION_NOTICE />

The `reset` operation resets the synchronization service, as well as restarts both the synchronization and compute platform services. The reset operation should not be run if there are outstanding GeoAnalytics jobs being performed, and no analysis using the GeoAnalytics Server can be performed during the reset. Once the reset has completed, you will need to wait five minutes before checking the status of your synchronization and compute platform services.

