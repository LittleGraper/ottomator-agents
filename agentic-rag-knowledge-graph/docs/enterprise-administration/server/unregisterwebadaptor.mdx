## Description

The `unregister` operation unregisters a [Web Adaptor](/webadaptor), which removes it from the server's trusted list. Once unregistered, the Web Adaptor can no longer submit requests to the server.

## Example usage

The following is a sample POST request for the `unregister` operation:

```markdown
POST /webadaptor/admin/system/webadaptors/bb2873dd-a52c-49b2-9de6-4790f5c2e638/unregister HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
