## Description

The `handlers` resource provides access to the server's handlers. A handler exposes the GIS capabilities of ArcGIS Server through a specific interface/API. There are two types of handlers currently available in the server: 

- Rest—Exposes the REST-ful API
- Soap—Exposes the SOAP API

The [Rest Handler](/handlersrest) resource exposes some of the administrative operations on the REST handler such as clearing the cache.
