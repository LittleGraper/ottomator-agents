## Description

<Note type={"note"}>

Clusters have been deprecated. The use of clusters for new deployments is strongly discouraged and existing deployments are advised to migrate away from use of clusters as part of ongoing maintenance and upgrades.

</Note>

This operation removes [ server machines](/machine) from the cluster. The server machines are returned back to the pool of registered server machines.

## Request parameters

<StyledTable headers={"Parameter,Description"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`machineNames` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

A comma-separated list of machine names.

Example

```javascript
machineNames=MACHINE2.DOMAIN.COM,MACHINE3.DOMAIN.COM
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is `html` .

Values: `html`  \| `json`  \| `pjson` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

Below is a sample POST request for `remove` :

```other
POST /webadaptor/admin/clusters/MapsCluster/machines/remove HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

machineNames=MACHINE2.DOMAIN.COM,MACHINE3.DOMAIN.COM&f=json
```
