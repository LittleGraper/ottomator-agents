## Description

The `soap` resource provides access to the `soaphandlersconfig` resource endpoint.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `soap` resource:

```markdown
https://machine.domain.com/webadaptor/admin/system/handlers/soap?f=pjson
```
