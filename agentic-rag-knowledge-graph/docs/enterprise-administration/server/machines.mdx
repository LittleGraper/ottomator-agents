## Description

The `machines` resource represents a collection of all the server machines that have been registered with the site; that is, it represents the total computing power of your site. A site will continue to run as long as there is one server machine online.

The list of server machines in your site can be dynamic. You can [register](/registermachine) additional server machines when you need to increase the computing power of your site or [unregister](/unregistermachine) them if you no longer need them.

## Example usage

The following is a sample request URL used to access the `machines` resource:

```markdown
https://machine.domain.com/webadaptor/admin/machines?f=pjson
```

  "machines": [ <machine1>, <machine2>, ...]
