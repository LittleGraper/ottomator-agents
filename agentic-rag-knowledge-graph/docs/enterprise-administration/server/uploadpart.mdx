## Description

The `uploadPart` operation uploads one part of an item. Before invoking this operation, you need to invoke the [Register Item](/registeritem) operation to reserve your item. After you have uploaded all the parts required to make the item, you need to consolidate all your parts by invoking the [Commit Item](/commititem) operation.
