## Description

The `unregister` operation unregisters all extensions from a previously registered server object extension (.soe) file. 


##  Request parameters

<StyledTable headers={"Parameter,Description"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `extensionFilename` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the previously registered .soe file.

      ```markdown label=Example
      extensionFilename=RouteFinder.soe
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`. 

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `unregister` operation:

```markdown
POST /webadaptor/admin/services/types/extensions/unregister HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

extensionFilename=RouteFinder.soe&f=json
```
