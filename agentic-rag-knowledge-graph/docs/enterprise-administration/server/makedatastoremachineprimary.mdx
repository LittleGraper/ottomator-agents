## Description

The `makePrimary` operation promotes a standby machine to the primary Data Store machine while also downgrading the existing primary machine to act as a standby machine.

## Example usage

The following is a sample POST request for the `makePrimary` operation:

```markdown
POST /webadaptor/admin/data/items/enterpriseDatabases/myDatastore/machines/DATASTOREMACHINE.DOMAIN.COM/makePrimary HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
