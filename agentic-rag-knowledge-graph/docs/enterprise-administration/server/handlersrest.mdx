## Description

The `rest` resource provides access to operations to clear the in-memory cache built by the REST handler, or to enable or disable the HTML browsing capabilities of the handler.

<Note type={"note"}>

Invocation of any operation under this resource will cause the REST handlers to be restarted on every server machine in the site.

</Note>

## Example usage

The following is a sample request URL used to access the `rest` resource:

```markdown
https://machine.domain.com/webadaptor/admin/system/handlers/rest?f=pjson
```
