## Description

The `edit` operation updates the usage report settings. The usage reports settings are applied to the entire site.


##  Request parameters

<StyledTable headers={"Parameter,Description"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `enabled` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      When usage reports are enabled (`true`), service usage statistics are collected and persisted to a statistics database. When usage reports are disabled (`false`), the statistics are not collected.

      Values: `true` | `false`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `samplingInterval` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Defines the duration (in minutes) for which the usage statistics are aggregated or sampled, in-memory, before being written out to the statistics database.

      ```markdown label=Example
      samplingInterval=30
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `maxHistory` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Represents the number of days after which usage statistics are deleted after the statistics database. If the `maxHistory` parameter is set to 0, the statistics are persisted forever. The default value is 0 (forever).

      ```markdown label=Example
      maxHistory=0
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `edit` operation:

```markdown
POST /webadaptor/usagereports/settings/edit HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

enabled=true&samplingInterval=30&maxHistory=0&f=pjson
```

## JSON Response syntax

```other
{
  "settings": {
    "enabled": true,
    "samplingInterval": 30,
    "maxHistory": 0
  }
}
```
