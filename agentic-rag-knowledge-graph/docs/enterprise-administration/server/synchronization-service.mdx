## Description

<GAX_DEPRECATION_NOTICE />

The resource for the syncrhonization service provides access to operations to manage your synchronization service for ArcGIS GeoAnalytics Server, such as starting or stopping the service, or viewing its status. 

<Note>

While a health check operation is exposed for the synchronization service, it will not return any valuable information.

</Note>
