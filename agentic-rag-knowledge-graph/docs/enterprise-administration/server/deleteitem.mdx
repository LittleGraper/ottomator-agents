## Description

The `delete` operation deletes the uploaded item and its configuration.

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f`

  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `delete` operation:

```markdown
POST /webadaptor/admin/uploads/if4582d33-f10a-41fe-85bc-cf2e2880c46e/delete HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
