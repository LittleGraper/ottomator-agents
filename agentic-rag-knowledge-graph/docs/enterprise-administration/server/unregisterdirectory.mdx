## Description

The `unregister` operation unregisters a server directory. Once a directory has been unregistered, it can no longer be referenced (used) from within a GIS service.

<Note type={"note"}>

The contents of the directory are not deleted.

</Note>

## Example usage

The following is a sample POST request for the `unregister` operation:

```markdown
POST /webadaptor/admin/system/directories/arcgisjobs/unregister HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
