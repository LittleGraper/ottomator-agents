## Description

The `jobs` resource is a collection of all the administrative [jobs](/job) (asynchronous operations) created within your site. When operations that support asynchronous execution are run, the server creates a new job entry that can be queried for its current status and messages.

<Note type={"note"}>

These administrative jobs are not the same as geoprocessing jobs.

</Note>

## Example usage

The following is a sample request URL used to access the `jobs` resource:

```markdown
https://machine.domain.com/webadaptor/admin/system/jobs?f=pjson
```

## JSON Response syntax

```javascript
{
  "jobs": ["<jobID1>", "<jobID2>", ...]
}
```
