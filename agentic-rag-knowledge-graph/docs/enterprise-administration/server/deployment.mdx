## Description

<Note type={"note"}>

Sites with multiple clusters have been deprecated. The use of multi-cluster sites for new deployments is strongly discouraged and existing deployments are advised to migrate away from use of multi-cluster sites as part of ongoing maintenance and upgrades.

</Note>

The `deployment` resource returns information on the current deployment configuration. These properties can affect side-wide behavior.

## Example usage

The following is a sample request URL used to access the `deployment` resource:

```javascript
https://machine.domain.com/webadaptor/admin/system/deployment?f=pjson
```
