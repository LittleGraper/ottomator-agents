## Description

<GAX_DEPRECATION_NOTICE />

The `start` operation runs your synchronization service.

## Example usage

The following is a sample POST request for the `start` operation:

```markdown
POST /webadaptor/admin/system/platformservices/11c76edb-c46d-4b1f-981a-d3d8a0654604/start HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
