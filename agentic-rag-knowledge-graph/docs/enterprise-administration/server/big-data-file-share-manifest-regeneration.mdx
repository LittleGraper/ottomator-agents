## Description

<GAX_DEPRECATION_NOTICE />

The `regenerate` operation regenerates the manifest for a big data file share. A manifest should be regenerated when you have added new data or if you have uploaded a hints file using the edit resource.

<Note type={"caution"}>

When a manifest is regenerated, it will update the manifest for existing datasets that have a hints file or new datasets. Any edits you have made to the manifest will be overwritten with the rules defined in the hints file.

</Note>
