## Description

The `clear` operation clears the cache on all REST handlers in the system. While the server typically manages the REST cache for you, use this operation to get explicit control over the cache.

<Note type={"note"}>

Invoking this operation will cause REST handlers to be restarted on all server machines.

</Note>

## Example usage

The following is a sample GET request for the `clear` operation:

```markdown
https://machine.domain.com/webadaptor/admin/system/handlers/rest/cache/clear?f=pjson
```
