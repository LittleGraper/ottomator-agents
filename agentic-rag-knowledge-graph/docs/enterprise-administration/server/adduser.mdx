## Description

The `add` operation adds a user account to the [user store](/userstore). This is available only when the user store is a read-write store, such as the default ArcGIS Server store. If the username for the account you're attempting to add already exists in the user store, an error will be returned.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `username` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the user. The name must be unique in the user store.

      ```markdown label=Example
      username=jwhite
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `password` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The password for this user.

      ```markdown label=Example
      password=secret
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `fullname` 

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The full name for the user.

      ```markdown label=Example
      fullname=John White
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `description` 

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Adds comments or description for the user account.

      ```markdown label=Example
      description=A platinum customer
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `email` 

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      An email for the user account.

      ```markdown label=Example
      email=jwhite@email.com
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `add` operation:

```markdown
POST /webadaptor/admin/security/users/add HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

username=jwhite&password=secret&fullname=John White&description=A platinum customer&email=jwhite@email.com&f=pjson
```
