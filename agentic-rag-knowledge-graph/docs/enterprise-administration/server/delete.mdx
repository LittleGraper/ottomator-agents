## Description

The `delete` operation deletes a service. When a service is deleted, it is [stopped](/stopservice) and all resources and configurations associated with the service are also deleted.


## Request parameters

<StyledTable headers={"Parameter,Description"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

    `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`. 

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The folowing is a sample POST request for the `delete` operation:

```markdown
POST /webadaptor/admin/services/Map/testMap.MapServer/delete HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
