## Description

The `extensions` resource is a collection of all the custom server object extensions that have been uploaded and registered with the server. You can register new server object extensions using the [Register Extension](/registerextension) operation. When updating an existing extension, you need to use the [Update Extension](/updateextension) operation. If an extension is no longer required, you can use the [Unregister](/unregisterextension) operation to remove the extension from the site.
