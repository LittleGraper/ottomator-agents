## Description

The `settings` resource returns the current log settings.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `settings` resource:

```markdown
https://machine.domain.com/webadaptor/admin/logs/settings?f=pjson
```

## JSON Response example

```javascript
{
  "settings": {
    "logDir": "C:\\arcgisserver\\logs\\"
    "logLevel": "INFO",
    "maxLogFileAge": 90,
    "maxErrorReportsCount": 10
  }
}
```
