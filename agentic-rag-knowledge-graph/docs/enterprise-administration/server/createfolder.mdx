## Description

The `createFolder` operation creates a sub-folder underneath the root folder. ArcGIS Server supports only a single level hierarchy of folders and hence this operation is only supported on the root folder. A new folder inherits the permissions of the root folder during creation time. You can change the permissions on the folder by using the [Permissions](/permissions) resource on the created folder resource.

## Example usage

The following is a sample POST request for the `createFolder` operation:

```javascript
POST /webadaptor/admin/services/createFolder HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

folderName=Maps&description=A folder for map services&f=pjson
```
