## Description

The `logs` resource provides access to operations and resources associated with the server logs. Logs are the records written by the various components of ArcGIS Server. You can [query](/logsquery) the logs and change various [log settings](/logssettings).

## Example usage

The following is a sample request URL used to access the `logs` resource:

```javascript
https://machine.domain.com/webadaptor/admin/logs?f=pjson
```
