## Description

The `System` folder resource contains automatically deployed services used by ArcGIS Server. The following table outlines the services included in the `System` folder.

<StyledTable headers={"Service,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `CachingControllers` 
    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `CachingControllers` service is used by ArcGIS Server to create and manage map, globe and image service caches.

  </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `CachingTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `CachingTools` service is used by ArcGIS Server to create and manage map and image service caches.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `DistributedWorker` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A work item is opened on a machine in an ArcGIS Server cluster on behalf of a distributed geoprocessing tool.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `DynamicMappingHost` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      This service is a multitenant map service provider.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `FeatureServiceTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `FeatureServiceTools` service is used by a feature service to run various tools, for example, `CalculateFeatureService`.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `GeoAnalyticsManagement` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `GeoAnalyticsManagement` service is used for the management of GeoAnalytics.

      <Note type="legacy">

        The ArcGIS GeoAnalytics Server extension is
        [deprecated](https://support.esri.com/en-us/knowledge-base/deprecation-arcgis-geoanalytics-server-000032771)
        with the release of ArcGIS Enterprise 11.4. The last ArcGIS Enterprise release with support for GeoAnalytics Server
        is 11.3.

      </Note>

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `GeoAnalyticsTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `GeoAnalyticsTools` service is provided for distributed analysis of large datasets.

      <Note type="legacy">

        The ArcGIS GeoAnalytics Server extension is
        [deprecated](https://support.esri.com/en-us/knowledge-base/deprecation-arcgis-geoanalytics-server-000032771)
        with the release of ArcGIS Enterprise 11.4. The last ArcGIS Enterprise release with support for GeoAnalytics Server
        is 11.3.

      </Note>

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `LocationReferencingSystemTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `LocationReferencingSystemTools` service is used for supporting asynchronous location referencing capabilities.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `OrthomappingTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `OrthomappingTools` service is used by ArcGIS Server to provide distributed bundle block adjustment.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `ParcelFabricTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `ParcelFabricTools` service is used for supporting asynchronous parcel fabric capabilities.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `PublishingTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `PublishingTools` service is used by ArcGIS Server to publish service definition files as services.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `RasterAnalysisTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `RasterAnalysisTools` service is used by ArcGIS Server to provide distributed raster analysis.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `RasterProcessing` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `RasterProcessing` service is used by ArcGIS Server to process raster data.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `RasterProcessingGPU` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `RasterProcessingGPU` service is used by ArcGIS Server to process raster data using GPU.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `RasterRendering` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `RasterRendering` service is a multitenant raster rendering provider.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `RealityMappingTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `RealityMappingTools` eoprocessing service is used by ArcGIS Server to support reality mapping workflows using the nFrames SURE engine.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `ReportingTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `ReportingTools` service is used by ArcGIS Server to report the cache status for map and image service caches.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `SceneCachingControllers` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `SceneCachingControllers` service is used by ArcGIS Server to create and manage scene service caches.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `SceneCachingTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `SceneCachingTools` service is used by ArcGIS Server to create and manage scene service caches.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `SpatialAnalysisTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `SpatialAnalysisTools` service is used for supporting spatial analysis capabilities in Portal for ArcGIS.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `SyncTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `SyncTools` service is used by a feature service to synchronize your organization's GIS content.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `TopographicProductionSystemTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `TopographicProductionSystemTools` service is used for supporting asynchronous topographic production capabilities.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `UtilityNetworkTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `UtilityNetworkTools` service is used for supporting asynchronous utility network capabilities.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `ValidationTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `ValidationTools` service is used for supporting asynchronous validation capabilities.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `VersionManagementTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `VersionManagementTools` service is used for supporting asynchronous version management capabilities.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `System` folder resource:

```javascript
https://machine.domain.com/webadaptor/admin/services/System?f=pjson
```
