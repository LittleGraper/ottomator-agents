## Description

The `Utilities` folder contains automatically deployed services that can be used by application developers. The table below outlines the services included in the `Utilities` folder.

<StyledTable headers={"Service,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `GeocodingTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `GeocodingTools` aervice is used for supporting asynchronous batch geocoding capability in Portal for ArcGIS.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `GeometryServer` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `Geometry` service is used for performing geometric calculations to support querying, analysis, and editing operations from client applications.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `OfflinePackaging` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `OfflinePackaging` service is used to create and set up map areas. Map areas make it faster for your users to work offline by packaging your map for a specific area ahead of time.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `PrintingTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `PrintingTools` service is used to export the contents of a web map to PDF, JPEG, PNG. or other format. This service is generally used in the context of providing applications with printing functionality.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `RasterUtilities` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `RasterUtilities` service contains tools that enable raster analysis and processing.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `SymbolServer` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The `Symbol` service is used to perform symbol-related functions.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `Utilities` folder resource:

```markdown
https://machine.domain.com/webadaptor/admin/services/Utilities?f=pjson
```
