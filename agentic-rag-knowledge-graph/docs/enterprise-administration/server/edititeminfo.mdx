## Description

The `edit` operation modifies the service's metadata.


##  Request parameters

<StyledTable headers={"Parameter,Description"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `iteminfo` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A JSON object representing the metadata.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `thumbnail` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      An optional thumbnail image to be uploaded to the server.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
