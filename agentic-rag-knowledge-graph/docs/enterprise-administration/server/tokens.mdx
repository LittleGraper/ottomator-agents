## Description

The `tokens` resource represents the token configuration associated with the server. You can use the [Update Token Configuration](/updatetokenconfig) operation to change the shared secret key or the valid token durations.


## Request parameters

<StyledTable headers={"Parameter,Description"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `tokens` resource:

```markdown
https://machine.domain.com/webadaptor/admin/security/tokens?f=json
```

## JSON Response example

This resource returns the current [token manager configuration](/tokenmanager).

```javascript
{
  "type": "BUILTIN",
  "properties": {
    "sharedKey": JQDMe5Idm+8opD68HKGM8hvBngmOuceNVE9a2y/KLR=,
    "longTimeout": "2880",
    "shortTimeout": "120",
  }
}
```
