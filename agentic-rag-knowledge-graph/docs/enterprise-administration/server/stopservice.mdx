## Description

The `stop` operation stops all instances of the service on all reachable [server machines](/machine) registered with the site. If a service is stopped, it cannot service any incoming requests.


## Request parameters

<StyledTable headers={"Parameter,Description"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `stop` operation:

```markdown
POST /webadaptor/admin/services/Maps/SeattleMap.MapServer/stop HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
