## Description

The `enable` operation enables login privileges for the primary site administrator account. This operation can only be invoked by an another administrator in the system. However, if you did not have any other administrators in the system and accidentally disabled the primary site administrator account, you can re-enable the account by running the password reset utility. This utility is shipped in `<ArcGIS Server installation directory>\Server\tools\passwordreset`. Use the `-e` option to re-enable the primary site administrator. For more information on this utility, see [Resetting a forgotten password](https://enterprise.arcgis.com/en/server/latest/administer/windows/resetting-a-forgotten-password.htm).
