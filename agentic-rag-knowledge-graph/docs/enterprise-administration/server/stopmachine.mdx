## Description

The `stop` operation stops the application server running on the specific server machine. When a server machine is stopped, it can no longer run GIS [services](/services). As a result, the total capacity of the server site is reduced.

<Note type={"note"}>

The server will continue to process administrative commands. To completely stop the server, you need to stop the Windows Service or use the `stopserver` script on Linux.

</Note>

## Example usage

The following is a sample POST request for the `stop` operation:

```markdown
POST /webadaptor/admin/machines/MACHINE.DOMAIN.COM/stop HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
