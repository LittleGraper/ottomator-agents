## Description

<Note type={"note"}>

As of 10.7, this service is no longer available. Support for the Synchronization Service health check is available in ArcGIS Enterprise 10.5.x and 10.6.x.

</Note>

The health check returns information for each machine that participates in the Synchronization Service. While this operation is exposed for the synchronization service, it will not return any valuable information.
