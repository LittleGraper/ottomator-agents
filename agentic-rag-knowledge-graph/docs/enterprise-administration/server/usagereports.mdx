## Description

The `usagereports` resource is a collection of all the usage reports created within your site. The [Create Usage Report](/createusagereport) operation lets you define a new usage report.


##  Request parameters

<StyledTable headers={"Parameter,Description"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `usagereports` resource:

```markdown
https://machine.domain.com/webadaptor/admin/usagereports?f=pjson
```

## JSON Response syntax

```other
{ 
   "metrics": [<usage-report1>, < usage-report2>] 
}
```
