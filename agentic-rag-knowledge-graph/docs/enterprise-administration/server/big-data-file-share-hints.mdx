## Description

<GAX_DEPRECATION_NOTICE />

The `hints` resource returns the hints resource for a big data file share. Hints are advanced parameters to control the generation of a manifest. 


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
  <StyledTableCell cellStyle="markdown">

      `download` 

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      This will download hints as a hints.dat file.

      Values: `true` 

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `read` 

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      This will return the content of the hints file in text/plain format.

      Values: `true` 

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`. 

      Values: `html` 

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
