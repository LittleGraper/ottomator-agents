## Description

This `servers` resource returns detailed information about an ArcGIS Server registered with an organization, which includes the following information: 

- The server's ID
- The server's name
- The ArcGIS Web Adaptor URL
- The server's administrative URL
- Whether the server is set as a hosting server
- The server role
- The server type

## Example usage

The following is a sample URL used to access the `servers` resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/federation/servers?f=pjson
```
