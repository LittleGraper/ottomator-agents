## Description

<Note type={"note"}>

This endpoint has been deprecated with the release of ArcGIS Enterprise 11.2. Starting at 11.2, workflows should be updated to use the new [External Content](/external-content) resource and the [Update](/update-external-content) operation.

</Note>

The `configuration` resource provides access to the [update content configuration](/update-content-configuration) operation and returns if external content discovery has been enabled.

## Example usage

The following is a sample request URL used to access the `configuration` resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/system/content/configuration?f=pjson
```

## JSON Response syntax

```javascript
{"isExternalContentEnabled": <true|false>}
```

## JSON Response example

```javascript
{"isExternalContentEnabled": true}
```
