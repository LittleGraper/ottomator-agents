## Description

The `getAppInfo` operation retrieves the client ID, and a list of its associated redirect URIs, for a specific application registered with an organization. The returned information can then be used to [update the app's current redirect URIs](/update-app-info).

##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `appID` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The client ID of the application.

      ```markdown label=Example
      appID=mapsForSAPBObj
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample GET request for the `getAppInfo` operation:

```markdown
https://machine.domain.com/webadaptor/portaladmin/security/oauth/getAppInfo?appID=mapsForSAPBObj&f=json
```

{
  "appId": "mapsForSAPBObj",
  "redirectURIs": [
    "https://la.arcgis.com",
