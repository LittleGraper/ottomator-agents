## Description

The `machines` resource lists all machines registered with the portal site. The response also returns the information for each machine, such as each machine's admin URL and its OS platform information. Each machine can have its [status](/status) polled to check whether the machine is ready to accept requests.

## Example usage

The following is a sample request URL used to access the `machines` resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/machines?f=pjson
```

## JSON Response syntax

```other
{
  "machines": [
    {
      "machineName": "<machine name>",
      "adminURL": "<admin URL>",
      "role": "<standby|primary>",
      "platform": "<OS platform information>"
    }
  ]
}
```
