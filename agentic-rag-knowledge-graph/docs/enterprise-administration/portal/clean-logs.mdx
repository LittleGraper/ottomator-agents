## Description

The `clean` operation deletes all log files stored by the machine hosting ArcGIS Enterprise portal, freeing up disk space. This includes audit logs, which were introduced at 11.4.

Deleted logs cannot be recovered after performing this operation.

## Example usage

The following is a sample POST request for the `clean` operation:

```markdown
POST /webadaptor/portaladmin/logs/clean HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
