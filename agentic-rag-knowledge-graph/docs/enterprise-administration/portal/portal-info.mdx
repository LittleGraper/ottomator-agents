## Description

The `info` resource returns the current version and build information for ArcGIS Enterprise portal.

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

  </StyledTableRow>

The following is a sample request URL used to access the `info` resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/info?f=pjson
```
