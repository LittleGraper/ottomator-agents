## Description

The resource for an individual machine returns information on a specific portal machine registered with ArcGIS Enterprise portal, such as the administrative URL for the machine and its OS platform information. A portal machine represent a machine on which Portal for ArcGIS software has been installed and licensed. From this resource, you can review the [status](/status) of the machine and manage SSL certificates when in a highly available portal environment.

## Example usage

The following is a sample request URL used to access an individual machine resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/machines/machine.domain.com?f=pjson
```

## JSON Response syntax

```other
{
  "machineName": "<machine name>",
  "adminURL": "<admin URL>",
  "role": "<standby|primary>",
  "platform": "<OS platform information>"
}
```
