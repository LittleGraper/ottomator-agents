## Description

The `generateCsr` operation generates a certificate signing request (CSR) for a self-signed certificate. A CSR is required by a CA to create a digitally signed version of your certificate.

<Note type={"note"}>

This operation is not accessible for highly available portals. In such instances, a counterpart [Generate CSR](/generate-csr-ha) operation should be used in its place. For more information on high availability, see [Configure a highly available portal](https://enterprise.arcgis.com/en/portal/latest/administer/windows/configuring-a-highly-available-portal.htm).

</Note>
