## Description

The `settings` resource returns whether incremental backups have been enabled or disabled for your organization. If enabled (`true`), the [Edit](/edit-database-settings) operation can be used to disable (`false`) incremental backups. The `edit` operation cannot, however, be used to enable incremental backups. For information on how to enable this property using the webgisdr utility, see [Create an ArcGIS Enterprise backup](https://enterprise.arcgis.com/en/portal/latest/administer/windows/create-web-gis-backup.htm) and [Configure disaster recovery for ArcGIS Enterprise](https://enterprise.arcgis.com/en/portal/latest/administer/windows/configure-disaster-recovery.htm).

```markdown

```
