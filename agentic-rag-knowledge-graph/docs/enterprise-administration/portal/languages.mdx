## Description

<Note type={"note"}>

This endpoint has been deprecated with the release of ArcGIS Enterprise 11.2. Starting at 11.2, workflows should be updated to reference the higher-level [Languages](/content-languages) resource and the [Add](/content-add-language) and [Remove](/content-remove-languages) operations.

</Note>

The `languages` resource lists which languages will appear in portal content search results. The [Update Languages](/update-languages) operation can change which language's content will be available, though disabling all languages is not supported. A minimum of one language must be enabled.

<Note type={"note"}>

Once [external content](/content) has been disabled, the languages resource will also be disabled. To enable languages, you must enable external content.

</Note>
