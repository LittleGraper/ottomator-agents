## Description

The `system` resource is an umbrella for a collection of system-wide resources for your portal.

##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>

## Example usage

The following is a sample request URL used to access the `system` resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/system?f=json
```
