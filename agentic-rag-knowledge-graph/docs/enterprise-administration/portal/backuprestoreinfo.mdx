## Description

The `backupRestoreInfo` resource displays the backup and restore information for an organization, such as when the last full or incremental backups were performed, as well as when the last full or incremental restores were performed, using the `webgisdr` utility. Starting at ArcGIS Enterprise 11.1, this resource also returns the date and time the last backup was performed in backup mode. For more information about these modes, see [ArcGIS Enterprise backups](https://enterprise.arcgis.com/en/portal/latest/administer/windows/overview-backup-restore-web-gis.htm).

    <StyledTableCell cellStyle="markdown">

      `backupModeTimeStamp`

    <StyledTableCell cellStyle="markdown">

    <StyledTableCell cellStyle="markdown">

## Example usage

The following is a sample request URL used to access the `backupRestoreInfo` resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/backupRestoreInfo?f=pjson
```
