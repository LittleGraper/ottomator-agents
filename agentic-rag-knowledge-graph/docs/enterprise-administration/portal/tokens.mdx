## Description

The `tokens` resource represents the token configuration for an organization. Use the [Update Token Configuration](/update-token-configuration) operation to change the configuration properties of the token service.

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>

## Example usage

The following is a sample request URL used to access the `tokens` resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/security/tokens?f=pjson
```
