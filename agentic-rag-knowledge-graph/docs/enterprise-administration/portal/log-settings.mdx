## Description

The `settings` resource returns the current log settings for ArcGIS Enterprise portal.

##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>

## Example usage

The following is a sample request URL used to access the `settings` resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/logs/settings?f=pjson
```

## JSON Response example

```other
{
  "settings": {
    "logDir": "C:\\arcgisportal\\logs\\",
    "logLevel": "INFO",
    "maxLogFileAge": 90
  }
}
```
