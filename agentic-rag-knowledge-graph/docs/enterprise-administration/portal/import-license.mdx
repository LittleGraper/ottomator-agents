## Description

The `importLicense` operation is used to import a new license file. The license file contains your ArcGIS Enterprise portal's user types and add-on licenses. By importing a license file, you are applying the licenses to your organization.

<Note type={"caution"}>

Importing a new license file will overwrite your organization's current user type and add-on licenses. Before importing, verify that the new license file has sufficient user type, app, and app bundle licenses.

</Note>
