## Description

<Note type={"note"}>

This endpoint has been deprecated with the release of ArcGIS Enterprise 11.2. Starting at 11.2, workflows should be updated to reference the [Languages](/content-languages) resource and the [Add](/content-add-language) and [Remove](/content-remove-languages) operations.

</Note>

The `update` operation updates the languages that will have content displayed in portal search results. When called, this operation triggers a reindex of the portal. This may take awhile to finish. During this time, the index service may not respond correctly to index queries.

##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `languages` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Lists, as form data, the language values for languages that will be displayed in content search results. For a complete list of portal languages, see the [Language values](/update-languages#language-values) table below.

      ```other label=Example
      languages=esri_ar
      ```

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
