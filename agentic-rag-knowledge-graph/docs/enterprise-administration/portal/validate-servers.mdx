## Description

The `validate` operation returns information on the status of the ArcGIS Server types registered with an organization. Upon completion, this operation returns status and accessibility information for all servers. This response also includes any failure messages from failed validation checks. Starting at ArcGIS Enterprise 11.1, the response includes the `adminUrl`, which returns the administrative URL for each federated server.

## Example usage

The following is a sample GET request for the `validate` operation:

```markdown
https://machine.domain.com/webadaptor/portaladmin/federation/servers/validate?f=pjson
```

## Response values

The following list outlines the possible `status` values included in the operation's response:

-   `success` 
-   `success with warnings`
-   `failure` 
-   `undetermined`
