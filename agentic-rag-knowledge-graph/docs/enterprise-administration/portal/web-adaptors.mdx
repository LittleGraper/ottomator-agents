## Description

The `webadaptors` resource lists the ArcGIS Web Adaptor configured with a portal machine. You can configure the ArcGIS Web Adaptor by using its configuration web page or the command line utility provided with the installation.

##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>

## Example usage

The following is a sample request URL used to access the `webadaptors`  resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/system/webadaptors?f=json
```

 

## JSON Response syntax

```other
{
  "webAdaptors": [Web Adaptor]
}
```
