## Description

The `importSignedCertificate` operation imports a certificate authority (CA) signed SSL certificate into the key store.

<Note type={"note"}>

This operation is not accessible for highly available portals. In such instances, a counterpart [Import Signed Certificate](/import-signed-certificate-ha) operation should be used in its place for a high availability workflow. For more information on high availability, see [Configure a highly available portal](https://enterprise.arcgis.com/en/portal/latest/administer/windows/configuring-a-highly-available-portal.htm).

</Note>
