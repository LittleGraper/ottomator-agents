## Description

<Note type={"note"}>

Prior to ArcGIS Enterprise 11.2, the [Update Languages](/update-languages) operation was used to enable or disable the languages that can display content in portal search results.

</Note>

The `remove` operation disables currently enabled languages for your organization. Once disabled, the language's related content will no longer be accessible to users in your organization and will be listed as disabled (`false`) in the Languages resource response. Disabled languages can be reenabled using the [Add](/content-add-language) operation.

##  Request parameters 

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `languages` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A comma-separated list of language codes for the languages that will be disabled.

      <Note type={"note"}>

      An organization must have a minimum of one language enabled at any time. Attempting to remove all enabled languages will result in an error.

      </Note>

      ```javascript label=Example
      languages=fr,de
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
