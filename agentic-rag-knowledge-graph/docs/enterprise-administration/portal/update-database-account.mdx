## Description

The `updateAdminAccount` operation changes the database administrator account credentials. By default, the initial administrator account you define during the [Create Site](/create-site) operation is used as the database administrator account. To change just the password for the account, provide the `password` parameter. If you want to create a completely new account for the database, provide new values for the `username` and `password` parameters.

<Note type={"note"}>

Updating the credentials for the database administrator account will cause both the database and the portal to be restarted.

</Note>

## Example usage

The following is a sample POST request for the `updateAdminAccount` operation:

```javascript
POST /webadaptor/portaladmin/system/database/updateAdminAccount HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

username=newdbadmin&password=newdbadminpassword&f=pjson
```
