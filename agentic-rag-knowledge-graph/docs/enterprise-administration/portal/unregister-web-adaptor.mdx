## Description

The `unregister` operation unregisters a ArcGIS Web Adaptor from a portal site. Once a web adaptor has been unregistered, the portal will no longer trust the web adaptor and will not accept any credentials from it. This operation is typically used when registering a new web adaptor or when an old web adaptor needs to be updated.

The following is a sample POST request for the `unregister` operation:

```javascript
POST /webadaptor/portaladmin/system/webadaptors/a62c3fed-50f9-4f15-83fa-aec7a4172af4/unregister HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

f=pjson
```
