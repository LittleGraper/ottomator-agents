      `alias` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the certificate.

      ```markdown label=Example
      alias=mycustomca
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `file` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The multipart POST parameter containing the certificate file.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `norestart` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Determines whether the portal should be prevented from restarting after importing the certificate. By default, this value is set as `false` and the portal will restart.

      Values: `true` | `false`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
