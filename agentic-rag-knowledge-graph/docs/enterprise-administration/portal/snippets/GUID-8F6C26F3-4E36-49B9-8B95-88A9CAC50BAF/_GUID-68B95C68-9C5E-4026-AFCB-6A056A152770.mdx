<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `alias` 

      (Required)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the certificate.

      ```markdown label=Example
      alias=myselfsignedcertificate
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `keyAlg` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The algorithm used to generate the key pairs. The default is `RSA`.

      ```markdown label=Example
      keyAlg=RSA
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `keySize` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The size of the key. The default is `1024`.

      ```markdown label=Example
      keySize=2048
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `sigAlg` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The algorithm used to sign the self-signed certificates. The default is derived from the `keyalg` parameter.

      Example

      ```markdown label=Example
      sigAlg=SHA256withRSA
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `cn` 

      (Required)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The common name used to identify the server for which the certificate is to be generated.

      ```markdown label=Example
      cn=SERVER.DOMAIN.COM
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `orgUnit` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The department in which this server resides.

      ```markdown label=Example
      orgUnit=development
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `org` 

      (Required)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the organization.

      ```markdown label=Example
      org=DOMAIN.COM
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `city` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the city.

      ```markdown label=Example
      city=Buffalo
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `state` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the state.

      ```markdown label=Example
      state=NY
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `country` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The two-letter abbreviation for the country.

      ```markdown label=Example
      country=US
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `validity` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The expiration time for the certificate in days. The default is `90`.

      ```markdown label=Example
      validity=120
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `san` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The certificate's Subject Alternative Name. This value can be a machine name or an IP address, but it must be prefixed by either `DNS:` or `IP:`. Multiple values are supported, and must be separated by a comma without any spaces.

      ```markdown label=Example
      san=DNS:SERVER.DOMAIN.COM
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
