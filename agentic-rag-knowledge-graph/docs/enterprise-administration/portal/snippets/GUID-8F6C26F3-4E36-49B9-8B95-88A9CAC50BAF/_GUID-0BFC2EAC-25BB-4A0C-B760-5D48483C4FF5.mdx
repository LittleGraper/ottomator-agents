<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `alias` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the certificate.

      ```markdown label=Example
      alias=mysslcert
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `password` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The password to unlock the file containing the certificate.

      ```markdown label=Example
      password=password
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `file` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The multipart POST parameter containing the certificate file.

      ```markdown label=Example
      file=mycertificate.pfx
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `importCertificateChain` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Introduced in 11.3. Specifies if the root or intermediate certificates included inside the existing certificate file should also be imported. By default, the parameter is set to `true`.
      
      Values: `true` | `false`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
