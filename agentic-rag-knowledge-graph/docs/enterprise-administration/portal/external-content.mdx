## Description

<Note type={"note"}>

Prior to ArcGIS Enterprise 11.2, enabling or disabling external content was handled through the [Update Content Configuration](/update-content-configuration) operation.

</Note>

The `externalcontent` resource returns whether access to external content has been enabled or disabled for an organization. If the resource returns `true`, an organization's Esri content will contain external URLs that reference sites and resources hosted outside of the organization. If the resource returns `false`, Esri content containing external URLs will be removed from the organization. Any Esri content remaining after external content is disabled does not contain external URLs.

If you are configuring ArcGIS Enterprise in an environment in which no internet connection is available, or internet access is prohibited, access to external content should be disabled to avoid discovering content containing inaccessible URLs to external sites. An organization's current `externalcontent` setting can be changed using the Update operation.

## Example usage

The following is a sample request URL used to access the `externalcontent` resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/system/content/externalcontent?f=pjson
```
