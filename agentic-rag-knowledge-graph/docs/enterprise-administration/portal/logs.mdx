## Description

The `logs` resource provides access to various operations and resources that allow you to query and clean logs, as well as edit the log settings.

##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `logs` resource:

```markdown
https://machine.domain.com/webadaptor/portaladmin/logs?f=pjson
```

## JSON Response example

```javascript
{
  "resources": [
    "settings"
  ],
  "operations": [
    "query",
    "clean"
  ]
}
```
