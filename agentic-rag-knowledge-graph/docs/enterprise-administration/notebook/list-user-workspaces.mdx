## Description

The `listUserWorkspaces` resource provides the name of the workspace directory (`/arcgis/home/workspacedirectoryname`) for the user initiating the request. If the user is an administrator of the Notebook Server, it will return all workspace directory names.


<Note>


The `listUserWorkspaces` resource can also be accessed by the `/dataaccess/listUserWorkspaces` endpoint. 


</Note>
