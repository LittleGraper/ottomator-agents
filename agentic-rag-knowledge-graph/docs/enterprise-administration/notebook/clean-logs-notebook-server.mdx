## Description

This operation deletes all the log files currently in the ArcGIS Notebook Server site. This is a irreversible operation. 

Doing so forces the server to clean the logs, thus freeing up disk space. However, it is not required that you invoke this operation, because ArcGIS Notebook Server periodically purges old logs.
