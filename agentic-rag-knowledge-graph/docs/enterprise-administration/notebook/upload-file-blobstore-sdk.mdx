## Description

The upload file (blobstore SDK) operation is a PUT method that can only be accessed through HTTP requests and that allows you to upload a single file to the workspace directory (`/arcgis/home` ) of the user making the request. By default, the file is uploaded to the root of the workspace directory. This operation requires the following request headers:

```javascript
Content-Type: application/octet-stream
Content-Length: contentLength
x-ms-blob-type: BlockBlob
x-ms-version: 2020-02-10
```

 

## Example usage

```javascript
https://notebookserver.domain.com/notebookserver/admin/dataaccess/notebookworkspace/prepare_hurricane_data.zip 
```

 

## Response example

```other
HTTP Status code 201
```
