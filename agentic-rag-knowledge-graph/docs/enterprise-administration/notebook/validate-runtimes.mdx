## Description

This operation validates that Docker images that are associated with runtimes are installed on each machine of the Notebook Server site.

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

</StyledTableRow>

</StyledTable>

## Example usage

```javascript
https://notebookserver.domain.com:11443/arcgis/admin/notebooks/runtimes/validateRuntimes
```
