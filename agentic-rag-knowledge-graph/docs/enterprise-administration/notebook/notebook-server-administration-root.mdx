## Description

This is the root resource for administering ArcGIS Notebook Server. Starting from this root, the ArcGIS Notebook Server environment is organized into a [hierarchy of resources and operations](/resource-hierarchy).

A version number is returned as a part of this resource.

After installation, ArcGIS Notebook Server can be configured using the [Create Site](/create-site) operation.
