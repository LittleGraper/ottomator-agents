## Description

This resource provides information about the machine in your ArcGIS Notebook Server site. 

You can update some of these properties using the [Edit Machine](/edit-machine) operation.

<Note type={"note"}>

The `configureState`  and `realTimeState`  properties have been deprecated as of ArcGIS Notebook Server 11.0.

</Note>
