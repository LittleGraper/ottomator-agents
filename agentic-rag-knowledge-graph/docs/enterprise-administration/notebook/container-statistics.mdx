## Description

This resource returns the memory (MB) and CPU resources used by a container at the time when this request is made.

##  Request parameters

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is `html` .

Values: `html`  \| `json`  \| `pjson` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

Below is a sample request URL for the `statistics`  resource:

```javascript
https://notebookserver.domain.com:11443/arcgis/admin/system/containers/<containerID>/statistics
```
