## Description

The Notebook Server info resource provides general information about the notebook server, such as current version and current runtime version. It also returns information on whether the notebook server is secured using token-based authentication, as well as the token services URL. 

##  Request parameters

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Values: `html`  \| `json`  \| `pjson` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

```javascript
https://notebookserver.domain.com:11443/arcgis/rest/info
```

 

## JSON Response syntax

```other
{
  "owningSystemUrl": "<owningSystemUrl>",
  "fullVersion": "<fullVersion>",
  "currentRuntimeVersion": "<currentRuntimeVersion>",
  "currentVersion": <currentVersion>,
  "authInfo": {
    "tokenServicesUrl": "<tokenServicesUrl>",
    "isTokenBasedSecurity": true|fase
  }
}
```

 

## JSON Response example

```other
{
  "owningSystemUrl": "https://myportal.domain.com/portal",
  "fullVersion": "10.9.0",
  "currentRuntimeVersion": "5.0",
  "currentVersion": 10.9,
  "authInfo": {
    "tokenServicesUrl": "https://myportal.domain.com/portal/sharing/rest/generateToken",
    "isTokenBasedSecurity": true
  }
}
```
