## JSON Response syntax

```javascript
{
    "fullVersion": "<software version number>",
    "loggedInUserPrivilege": "<your user privilege>",
    "loggedInUser": "<your user name>",
    "currentversion": "<version>",
    "currentbuild": "<build_number>",
    "supportedFonts": ["font1","font2","font3" ]
}

```
