## Description

This resource displays the current log settings for the ArcGIS Notebook Server site. You can [edit these settings](/edit-log-settings).

##  Request parameters

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is `html` .

Values: `html`  \| `json`  \| `pjson` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## JSON Response examples

### JSON Response on a Windows machine

```javascript
{
  "settings": {
    "logDir": "C:\\arcgisnotebookserver\\logs\\"
    "logLevel": "INFO",
    "maxLogFileAge": 90
  }
}

```

 

### JSON Response on a Linux machine

```javascript
{
  "settings": {
    "logDir": "/data/ags/arcgis/notebookserver/usr/logs/",
    "logLevel": "INFO",
    "maxLogFileAge": 90
  }
}
```
