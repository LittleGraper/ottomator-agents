## Description

The primary site administrator (`psa` ) resource allows you to configure the credentials of the primary site administrator account. This account is stored on the server and can be used to sign in to the server and perform administrative workflows. Use the [Update primary site administrator operation](/update-primary-site-administrator) to change the name and password for the account.
