## Description

This operation will upgrade your ArcGIS Notebook Server site, including its configuration store and sample notebooks. The operation will only become available when your site can be upgraded, and will become hidden after you upgrade.

The ArcGIS Notebook Server Configuration Wizard uses this operation if your site is able to be upgraded. If you prefer not to use the wizard, or are unable to use it, you can call this operation from a script.
