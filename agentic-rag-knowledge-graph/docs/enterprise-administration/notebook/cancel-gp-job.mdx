## Description

The `cancel`  operation is performed on an asynchronous [GP `job` ](/gp-job) resource. This operation results in the cancellation of a geoprocessing job. You can provide an argument to the `cancel`  operation as defined in the parameters table below.

##  Request parameters 

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

</StyledTableRow>

</StyledTable>
