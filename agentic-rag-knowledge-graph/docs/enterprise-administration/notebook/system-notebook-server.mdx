## Description

The System resource is a collection of server-wide resources in your ArcGIS Notebook Server site. In this resource, you can access information and perform operations pertaining to [licenses](/licenses), [Web Adaptors](/web-adaptors-notebook-server), [containers](/containers), [server properties](/server-properties), [directories](/server-directories-notebook), [Jobs](/jobs-notebook-server), and the [configuration store](/configuration-store-notebook).
