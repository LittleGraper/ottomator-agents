## Description

The `exports` resource returns a list of request IDs for previously run updates or upgrades that encountered failures or errors during the upgrade process.

<Note type={"note"}>

Request IDs are assigned to every upgrade or update that is performed.

</Note>


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `exports` resource:

```other
https://organization.domain.com/context/admin/system/upgrades/history/exports?f=pjson
```
