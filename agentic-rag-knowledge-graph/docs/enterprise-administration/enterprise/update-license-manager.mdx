## Description

<Note type="deprecated">

This endpoint has been deprecated with the release of ArcGIS Enterprise 11.4. Starting at 11.4, ArcGIS License Manager is no longer required for named user licensing for ArcGIS Pro, ArcGIS Drone2Map or ArcGIS AllSource.

</Note>

The `updateLicenseManager` operation updates the license server connection information for an ArcGIS Enterprise portal and registers a backup license manager for high availability. After changing the `licenseManagerInfo` parameter properties, Portal for ArcGIS automatically restarts to register changes and set up connections with the backup license manager.

<Note type={"note"}>

Backup license managers must be authorized to use the same license file as the ArcGIS Enterprise portal.

</Note>
