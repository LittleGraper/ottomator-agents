## Description

The `webadaptors` resource lists the ArcGIS Enterprise on Kubernetes Web Adaptor configured for your deployment. The web adaptor can be configured using the `config` operation.


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `webadaptors` resource:

```markdown
https://organization.domain.com/context/admin/system/webadaptors?f=pjson
```
