## Description

The resource for an individual web adaptor, returning the web adaptor's properties such as the HTTP and HTTPS ports.


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access an individual web adaptor resource:

```markdown
https://organization.domain.com/context/system/webadaptors/0df813ce-96bd-4202-923a-d12c63f9a1ef?f=pjson
```
