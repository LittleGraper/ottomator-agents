## Description

The `restoreDefaultTemplate` operation restores the default notebook template, which is used when creating new notebooks, back to the original template that comes standard with ArcGIS Enterprise on Kubernetes. This operation can be invoked if the default template has been previously updated to use a custom template.

```other

Content-Type: application/x-www-form-urlencoded
```
