## Description

The `update` operation modifies an existing run for a scheduled task.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `status` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The status of the run.

      Values: `scheduled` | `executing` | `succeeded` | `skipped` | `failed` | `submitfailed`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `result` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A result string for the run.

      ```markdown label=Example
      result=GP Jobs have been cleaned.
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
