## Description

The `certificates` resource provides access to child operations and resources that can be used to manage all the security certificates configured with an organization.


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `certificates` resource:

```markdown
https://organization.domain.com/context/admin/security/certificates?f=pjson
```
