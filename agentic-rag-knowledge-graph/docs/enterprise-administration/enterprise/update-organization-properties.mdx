## Description

The `updates` operation updates one or more organization properties. When updating your organization properties, all currently configured properties must be included in the request along with any new properties or changes to preexisting property values. Only new properties added to the JSON object, or properties that have had their values updated, will be modified by this operation.


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `properties` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A JSON object of organization properties. At 10.9.1, the only supported organization property is `privatePortalURL`.

      ```markdown label=Example
      properties={"privatePortalURL": "https://organization.domain.com/context"}
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Organization properties

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `privatePortalURL` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Specifies the internal URL that a federated ArcGIS Server should use to communicate with ArcGIS Enterprise on Kubernetes-hosted services. This property is typically used when ArcGIS Enterprise on Kubernetes is configured for web-tier authentication.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
