## Description

When you install an ArcGIS Enterprise on Kubernetes deployment for the first time, [you must create a new organization site](/configure). A site is a collection of resources and operations that are used to administer and manage your organization. Once the organization site has been created, the result is the root resource for administering your deployment and a collection of hierarchically organized resources: [organizations](/organizations), [services](/services), [uploads](/uploads), [datastores](/datastores), [system](/system), [security](/security), [logs](/logs), and [jobs](/jobs). Alternative views of the root page, such as `json` and `pjson` response formats, return the organization site's state and logs from its configuration. All views return the deployment version number.

## Example usage

The following is a sample request URL used to access the ArcGIS Enterprise Administrator API root page:

```other
https://organization.domain.com/context/admin?f=pjson
```
