## Description

The `delete` operation allows the owner of a notebook to delete one or more snapshots associated with that notebook item.

## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `itemId`

      (Required)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The item ID of the notebook to list snapshots from.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `resourceKeys`

      (Required)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      A comma-separated list of resource keys of snapshots to delete. 
        
      Example:

      ```other
      resourceKeys=snapshot-resourcekey1,snapshot-resourcekey2,snapshot-resourcekey3

      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default response format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `delete` operation:

```other
POST /context/admin/notebooks/snapshots/delete HTTP/1.1
Host: organization.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

itemId=2ff206ba534541428409e327d03588d1&resourceKeys=snapshot-0f146a4ba1964327b396b9be5f1042ed.json&f=pjson&token=<token>

```
