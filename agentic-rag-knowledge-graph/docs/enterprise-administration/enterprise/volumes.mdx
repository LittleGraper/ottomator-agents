## Description

The `volumes` resource returns the settings for PersistentVolumeClaim (PVC) volumes, which provide additional resources to a GIS service's pods. Each volume includes properties defined during the volume's creation, as well as auto-generated information provided after the volume was created (the `mountPath` and `id` properties).

This resource also provides access to the [Create](/create-volumes) operation, which administrators can use to create PVC volumes, and the [Configurations](/volumes-configurations) resource.

