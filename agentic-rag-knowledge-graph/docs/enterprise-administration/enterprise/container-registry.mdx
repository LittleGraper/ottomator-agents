## Description

The resource for a container registry returns information, including the login credentials used to access the registry (`username` and encrypted `password`), about the container registry. The username and password can be updated using the [Edit](/edit-container-registry) operation.

      `f`

## Example usage

The following is a sample request URL used to access an individual container registry resource:

```markdown
https://organization.domain.com/context/admin/system/containerregistries/enf2lr3i7g2ht43a0yja5?f=pjson
```
