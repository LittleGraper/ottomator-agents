## Description

The `settings` operation returns the currently configured disaster recovery settings. The disaster recovery settings can be edited using the [Update](/update-settings) operation.


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `settings` resource:

```markdown
https://organization.domain.com/context/admin/system/disasterrecovery/settings?f=pjson
```
