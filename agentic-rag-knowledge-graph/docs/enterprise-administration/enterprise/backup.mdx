## Description

The resource for an individual backup returns information about the specific backup in a deployment.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access an individual backup resource:

```markdown
https://organization.domain.com/context/admin/system/disasterrecovery/stores/default/backups/backup_first?f=pjson
```
