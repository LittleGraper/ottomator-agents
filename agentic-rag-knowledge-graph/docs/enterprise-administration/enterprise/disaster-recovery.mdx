## Description

The `disasterrecovery` resource allows an administrator to manage disaster recovery settings, which include the following:

- Configuring settings for backup and restore operations
- Registering backup stores
- Creating backups
- Restoring backups
- Getting the status of ongoing or completed backup or restore operations
