## Description

The `terminateDeployment` operation stops a running notebook deployment. Administrators can terminate notebook deployments owned by any user. Other users can only terminate notebook deployments they own.

## Example usage

The following is a sample POST request for the `terminateDeployment` operation:

```other
 POST /context/admin/notebooks/deployments/<deploymentId>/terminateDeployment HTTP/1.1
 Host: organization.domain.com
 Content-Type: application/x-www-form-urlencoded
 Content-Length: []
 
 f=pjson&token=<token>

```
