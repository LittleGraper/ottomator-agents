## Description

The `getServicesById` operation returns a list of services corresponding to the IDs included in the request.

## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `serviceIDs`

      (Required)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Filters services by their specific service IDs. Service IDs can be gathered using the [Find Service IDs](/find-service-id) operation. If you are using multiple IDs, each ID must be separated by a comma.

      ```other label=Example
      serviceIDs=stkk34t9aakg2egw7zg8h

      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `getServicesById` operation:

```other
POST /context/admin/services/getServicesById HTTP/1.1
Host: organization.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

serviceIDs=stkk34t9aakg2egw7zg8h&f=pjson&token=<token>
```
