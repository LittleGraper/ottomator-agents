## Description

The `download` operation downloads a previously uploaded item. There are no required parameters for this operation.

## Example usage

The following is a sample request URL for the `download` operation:

```markdown
https://organization.domain.com/context/admin/uploads/25daa6c7-83dc-4810-bd3b-bd3349d3c36a/download
```
