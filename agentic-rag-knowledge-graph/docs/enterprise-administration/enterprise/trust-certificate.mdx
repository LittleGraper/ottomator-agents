## Description

The resource for an individual certificate, returning the details of a configured trust certificate for an organization.


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access an individual trust `certificate` resource:

```markdown
https://organization.domain.com/context/admin/security/certificates/trust/cc4474766-b0f7-4495-af7e-12b9ce50a057?f=pjson
```
