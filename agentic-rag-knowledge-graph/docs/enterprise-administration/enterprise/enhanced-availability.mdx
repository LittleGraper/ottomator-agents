## Description

The `enhanced-availability` architecture profile is used in business and mission-critical production environments. This profile is designed for the highest level of availability, as it includes increased and expanded redundancy across critical pods. As a high-availability profile, `enhanced-availability` provides continued use and availability in the event of a failure. However, of the available profiles, it has the most hardware requirements. For more information, see [Architecture profiles](https://enterprise-k8s.arcgis.com/en/latest/deploy/architecture-profiles.htm).

The table below lists the pods with the highest number of replicas for the `enhanced-availability` profile. All other pods, which are not included in the table below, have one (1) replica.
