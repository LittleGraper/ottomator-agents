## Description

The `security` resource is a collection of resource and operation endpoints that can be used to modify the security configuration for an organization.

  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `security` resource:

```markdown
https://organization.domain.com/context/admin/orgs/0123456789ABCDEF/security?f=pjson
```
