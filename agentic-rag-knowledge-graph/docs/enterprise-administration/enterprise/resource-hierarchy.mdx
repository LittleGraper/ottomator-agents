## Deprecated layout

At 10.9.1, the Backup and Restore endpoint has been deprecated. The child operations and resources located under this endpoint have been moved elsewhere within the Disaster Recovery portion of the API. The layout below demonstrates the 10.9 organization for Disaster Recovery:

-   Disaster Recovery

    -   Stores

        -   _Register_
        -   Backup store

            -   _Unregister_
    -   Backup and Restore

        -   _Get Status_
        -   _Create Backup_
        -   Backups

            -   Backup

                -   _Delete Backup_
                -   _Restore Backup_
    -   Disaster Recovery Settings

        -   _Update Settings_
