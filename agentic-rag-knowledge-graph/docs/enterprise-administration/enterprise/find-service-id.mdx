## Description

The `findServiceIds` operation queries and returns a list of the services within ArcGIS Enterprise on Kubernetes that match the filter criteria. Results can be filtered by specifying the folder location, type, or provider of a service. These filters are optional; if no filter is applied, the IDs of every service is returned in the response.

## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `filterFolderName`

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Filters services by their folder location.

      ```other label=Example
      filterFolderName=Hosted

      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `filterServiceName`
      
      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Filters services by their name.

      ```other label=Example
      filterServiceName=Countries

      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `filterServiceType`
      
      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Filters services by their service type.

      ```other label=Example
      filterServiceType=FeatureServer

      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `filterProvider`
      
      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Filters services by their provider types.

      Values: `ArcObjects11` | `SDS`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample POST request for the `findServiceIds` operation that demonstrates filtering services by their service type (`MapServer`):

```other
POST /context/admin/services/findServiceIds HTTP/1.1
Host: organization.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

filterFolderName=&filterServiceName=&filterServiceType=MapServer&filterProvider=&f=pjson&token=<token>
```
