## Description

The `development` architecture profile is used in nonproduction environments, including those for testing and evaluation, and requires the least amount of hardware and resources. This profile prioritizes replicated pods for publishing tools and the private ingress controller, setting replicas for both pods at 2. All other pod replicas are set at 1. For more information, see [Architecture profiles](https://enterprise-k8s.arcgis.com/en/latest/deploy/architecture-profiles.htm).
