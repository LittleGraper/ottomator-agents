## Description

The `enable` operation enables a previously disabled task, setting its `taskState` to `active`.

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f`

    <StyledTableCell cellStyle="markdown">

  </StyledTableRow>
</StyledTable>
