## Description

The `enterprisefunctions` resource returns the premium capabilities that are enabled for an organization. Currently, administrators can enable raster analytics or image hosting if the organization has been licensed to use these premium capabilities. The `enterprisefunctions` resource also provides access to the [Add](/add-enterprise-functions) and [Remove](/remove-enterprise-functions) operations, which are used to enable or disable specific premium capabilities.
