## Description

A workspace resource returns information about the notebook workspace that is registered with your organization. The properties of a workspace includes properties defined during the workspace creation, as well as auto-generated information provided after the workspace was created (the `mountPath` and `id` properties).  This resource also provides access to the Status resource for checking the status of the workspace and the Delete operation that administrators can use to delete the notebook workspace.
