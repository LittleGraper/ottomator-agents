## Description

The `update` operation is used to change the common properties and configurations for the ArcGIS Enterprise on Kubernetes Web Adaptor configured with your deployment. The properties are stored in a JSON object. Therefore, every update must include all necessary properties.

<Note type={"note"}>

Any changes made using this operation will cause the web adaptor to be reconfigured. After performing this operation, you will need to reconfigure your web adaptor from its configuration page or using the command line.

</Note>


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `webAdaptorsConfig` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The JSON object containing all the properties for the web adaptor configuration. Only the `sharedKey` and `configureOrgUrl` properties are supported.

      ```markdown label=Example
      webAdaptorsConfig={
        "sharedKey": "4HRWWpCS5gSAn2hN2y1Z4fCIGwr7IfpxL0kVMCs4E2w=",
        "configureOrgUrl": "https://reverseproxy.esri.com/arcgis/admin/configure"
      }
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
