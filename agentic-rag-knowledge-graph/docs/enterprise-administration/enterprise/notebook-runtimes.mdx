## Description

The `runtimes` resource provides a list of registered notebook runtimes.

There is one runtime included in each container image, which is applied to a notebook deployment at launch time. Notebook authors open ArcGIS Notebooks in their individual notebook deployment. The runtime included with the container image makes a precise collection of Python modules available to each notebook in the notebook deployment.

By default, there are two runtimes: ArcGIS Notebook Python 3 Standard and ArcGIS Notebook Python 3 Advanced. These runtimes are loaded with their corresponding container images during ArcGIS Enterprise on Kubernetes deployment by enabling the `NotebookServer` enterprise function.
