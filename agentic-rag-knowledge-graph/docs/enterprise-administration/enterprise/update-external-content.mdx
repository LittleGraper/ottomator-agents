## Description

The `update` operation enables and disables access to Esri-provided content containing external URLs that reference sites and resources hosted outside of the organization. If external content is disabled, Esri content containing external URLs will be removed from the organization and will not be discoverable. This extends to any external content that would typically be included when [enabling new languages](/add-language) for your organization; that is, when access to external content is disabled, all external content included with any added languages is also disabled.

If you enable external content, Esri content containing external URLs is added to the organization and is discoverable. By default, external content is enabled. This extends to any external content included when enabling new languages for the organization, enabling access to external content included with added languages.

<Note type={"note"}>

For organizations upgrading to ArcGIS Enterprise on Kubernetes 11.0 from an earlier version, an administrator must manually set the `externalContentEnabled` property to disable external content after the upgrade. However, ArcGIS Enterprise on Kubernetes will honor the values set for `externalContentEnabled` when upgrading to later versions from 11.0.

</Note>


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `externalContentEnabled` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Specifies whether the organization will allow Esri-provided external content. If `true`, outside resources and sites will be accessible to users in your organization. If `false`, access to this content will be unavailable. The default value is `true`.

      Values: `true` | `false`

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
