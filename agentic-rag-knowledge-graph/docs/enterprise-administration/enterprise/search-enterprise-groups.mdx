## Description

The `serachEnterpriseGroups` operation searches groups in the configured enterprise role store. You can narrow the search using the `filter` parameter.

<Note type={"note"}>

This operation only supports enterprise role stores, such as LDAP, that have been configured using the [Update Identity Store](/update-identity-store) operation.

</Note>


## Request parameters

  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      (Optional)

    <StyledTableCell cellStyle="markdown">


    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `maxCount` 

      (Optional)

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The maximum number of records the client will accept.

      ```markdown label=Example
      maxCount=1000
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f`

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
