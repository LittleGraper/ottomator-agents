## Description

The `Utilities` folder contains automatically deployed `GeometryServer` and `GPServer` services that can be used by application developers. The table below outlines the services included in the `Utilities` folder.


<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `Geometry` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      This service is used by web applications to perform geometric calculations to support querying, analysis, and editing operations.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `PrintingTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      This service is used by web applications to export the contents of a web map to PDF, JPEG, PNG, or other formats. This service can be used to provide printing functionality to applications.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `OfflinePackaging` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      This service is used to create and set up map areas. Map areas make it easier and faster for your users to work offline by packaging a map for a specific area ahead of time.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `GeocodingTools` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      This service is used to support the asynchronous batch geocoding capability.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `Utilities` folder resource:

```markdown
https://organization.domain.com/context/admin/services/Utilities?f=pjson
```
