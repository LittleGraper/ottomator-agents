## Description

The `backups` resource returns the backups that have been created for an organization. These backups can be used to restore the deployment to a specific point in time. All services, content, groups, settings, and configurations are restored to what was captured when the backup was created.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `backups` resource:

```markdown
https://organization.domain.com/context/admin/system/disasterrecovery/stores/backupStore/backups?f=pjson
```
