## Description

The `settings` resource returns the current log settings for an organization. At ArcGIS Enterprise 11.2 on Kubernetes, this resource returns the maximum number of days logs are retained in the system before they are deleted (`maxLogAgeDays`).


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `settings` resource:

```other
https://organization.domain.com/context/admin/logs/settings?f=pjson
```

## JSON Response syntax

```other
{
  "maxLogAgeDays": <Number of days for the log retention policy>
  "logLevel": "<INFO | SEVERE | WARNING | FINE | VERBOSE | DEBUG>",
  "lastModified": <Milliseconds from epoch from when log settings were last updated>
}
```

## JSON Response example

```other
{
  "maxLogAgeDays": 60,
  "logLevel": "INFO",
  "lastModified": 1597076095038
}
```
