## Description

The resource for a notebook runtime returns the runtime's properties. The runtime is included in the container image, which is applied to a notebook deployment at launch time. Notebook authors open ArcGIS Notebooks in their individual notebook deployments. The runtime included with the container image makes a precise collection of Python modules available to each notebook in the notebook deployment.
