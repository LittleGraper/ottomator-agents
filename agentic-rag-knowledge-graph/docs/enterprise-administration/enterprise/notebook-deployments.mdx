## Description

The `deployments` resource returns a list of Kubernetes deployment objects, and their properties, that are present in the cluster for a user. An administrator can view and manage notebook deployments that are owned by all users. Non-administrator organization members will only see the notebook deployments they own.

<StyledTable headers={"Property,Details"}>
    <StyledTableRow>
        <StyledTableCell cellStyle="markdown">

          `notebookRuntimeVersion` 

        </StyledTableCell>
        <StyledTableCell cellStyle="markdown">

          The version of the notebook runtime used for this notebook deployment.

        </StyledTableCell>
    </StyledTableRow>
    <StyledTableRow>
        <StyledTableCell cellStyle="markdown">

          `notebookRuntimeName` 

        </StyledTableCell>
        <StyledTableCell cellStyle="markdown">

          The name of the notebook runtime used for this notebook deployment.

        </StyledTableCell>
    </StyledTableRow>
    <StyledTableRow>
        <StyledTableCell cellStyle="markdown">

          `dateCreated` 

        </StyledTableCell>
        <StyledTableCell cellStyle="markdown">

          When the notebook deployment was created.

        </StyledTableCell>
    </StyledTableRow>
    <StyledTableRow>
        <StyledTableCell cellStyle="markdown">

          `allowHiddenFileDownloads` 

        </StyledTableCell>
        <StyledTableCell cellStyle="markdown">

          By default, hidden files, files that have names starting with a dot (.), stored in a user's workspace cannot be downloaded using the data access download API. Set this property to `true` to allow the owner of those files to download them.

        </StyledTableCell>
    </StyledTableRow>
    <StyledTableRow>
        <StyledTableCell cellStyle="markdown">

          `id` 

        </StyledTableCell>
        <StyledTableCell cellStyle="markdown">

          The ID of the notebook deployment in the site.

        </StyledTableCell>
    </StyledTableRow>
    <StyledTableRow>
        <StyledTableCell cellStyle="markdown">

          `notebookRuntimeId` 

        </StyledTableCell>
        <StyledTableCell cellStyle="markdown">

          The ID of the notebook runtime used for this notebook deployment.

        </StyledTableCell>
    </StyledTableRow>
    <StyledTableRow>
        <StyledTableCell cellStyle="markdown">

          `username` 

        </StyledTableCell>
        <StyledTableCell cellStyle="markdown">

          The user the notebook deployment is assigned to. Only one user can be assigned any given container.

        </StyledTableCell>
    </StyledTableRow>
</StyledTable>

## Example usage

The following is a sample request URL used to access the `deployments` resource:


```other
https://organization.domain.com/context/admin/notebooks/deployments?f=pjson
```
