## Description

The `groups` resource is a collection of operations that inventory your organization's groups, such as [retrieving a list of users within a specific group](/get-users-within-enterprise-group) or [listing which groups a specific user is assigned to](/get-enterprise-groups-for-user). The JSON information for the `groups` resource returns the total number of enterprise groups in the system.
