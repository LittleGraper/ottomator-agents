## Description

The `containerimages` resource returns a list of the container images that have been pulled and used to deploy applications for ArcGIS Enterprise on Kubernetes.

## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f`

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to acess the `containerimages` resource:

```markdown
https://organization.domain.com/context/admin/system/containerimages?f=pjson

```
