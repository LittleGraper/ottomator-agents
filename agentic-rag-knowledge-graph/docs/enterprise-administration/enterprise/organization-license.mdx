## Description

The `license` resource returns high-level licensing details for your organization, such as the total number of registered members that can be added, and the current number of members in the organization. Information for all licenses associated with your organization is returned, some of which may not be available through the home application. It's recommended that you reference both the home application and the `license` resource to verify which licenses are associated with your organization.

<Note type={"note"}>

The ArcGIS Enterprise portal enforces user type licensing. Members are assigned a user type, which determines the privileges that can be granted to the member through a role. Each user type may include access to specific apps. Additionally, members can be assigned add-on applications and [user type extensions](https://enterprise-k8s.arcgis.com/en/latest/administer/license-user-type-extensions.htm).

</Note>

This resource also provides access to various operations that allow you to manage your organization's ArcGIS Enterprise portal licenses and capabilities.

<Note type="legacy">

At ArcGIS Enterprise 11.4, the `license` resource no longer returns ArcGIS License Manager information. ArcGIS License Manager is no longer required for named user licensing for ArcGIS Pro, ArcGIS Drone2Map or ArcGIS AllSource.

</Note>

## Example usage

The following is a sample request URL used to access the `license` resource:

```markdown
https://organization.domain.com/context/admin/orgs/0123456789ABCDEF/license?f=pjson
```
