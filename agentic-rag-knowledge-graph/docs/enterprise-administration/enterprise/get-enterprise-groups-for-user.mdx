## Description

The `getEnterpriseGroupsForUser` operation lists the groups assigned to a user account in the configured role store. You can use the `filter` parameter to narrow search results.


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `username` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The name of the user account for which to return their assigned groups.

      ```markdown label=Example
      username=domain\jdoe
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `filter` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      An optional wildcard string that can be used to filter a search result to return specific groups.

      ```other label=Example
      filter=*bldg*
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `maxCount` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The maximum number of results the client is willing to accept.

      ```markdown label=Example
      maxCount=10
      ```

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
