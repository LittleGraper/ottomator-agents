## Description

An individual `job` resource returns information about an asynchronous job, either currently running or completed, for a geoprocessing service.


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Response properties

<StyledTable headers={"Property,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `job_id` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The unique ID for the job.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `job_created` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The time, in milliseconds from epoch, the job was submitted.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `job_execution_start` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The time, in milliseconds from epoch, the job was started.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `job_execution_end` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The time, in milliseconds from epoch, the job was completed.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `url` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The URL of the job resource page.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `status` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The current status of the job.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `userName` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The user who submitted the job.

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
