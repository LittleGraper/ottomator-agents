## Description

The resource for an individual folder returns the folder's properties, including the folder's name and the services it contains.

<Note type={"note"}>

ArcGIS Enterprise on Kubernetes supports a single-level hierarchy of folders.

</Note>


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `folder` resource:

```other
https://organization.domain.com/context/admin/services/folderName?f=pjson

```

## JSON Response syntax

```other
{
  "folderName": "<folder_name>",
  "services": [<full_service_name1>, <full_service_name2>]
}
```
