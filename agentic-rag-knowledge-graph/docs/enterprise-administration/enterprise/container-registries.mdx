## Description

The `containerregistries` resource returns the container registries associated with your deployment. Currently, only one container registry per deployment is supported.


##  Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>


## Example usage

The following is a sample request URL used to access the `containerregistries` resource:

```markdown
https://organization.domain.com/context/admin/system/containerregistries?f=pjson
```
