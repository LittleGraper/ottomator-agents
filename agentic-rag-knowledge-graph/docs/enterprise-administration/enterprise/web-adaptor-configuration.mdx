## Description

The `config` resource is a collection of configuration properties that apply to the ArcGIS Enterprise on Kubernetes Web Adaptor configured with your deployment. The only supported property is `sharedKey`, which represents credentials that are shared with the web adaptor. The web adaptor will use these credentials to communicate with your deployment.
