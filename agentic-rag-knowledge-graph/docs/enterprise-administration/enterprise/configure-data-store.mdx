## Description

The `config` resource returns the current minimum and maximum resource allocations for a relational data store, including the minimum memory (`memoryMin`) and minimum CPU (`cpuMin`) resources. The data store's resources can be scaled using the [Edit](/update-datastore-configuration) operation.
