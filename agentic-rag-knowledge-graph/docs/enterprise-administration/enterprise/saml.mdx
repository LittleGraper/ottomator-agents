## Description

The `saml` resource returns information about the SAML configuration for an organization. If SAML is configured, the `enabled` property will return as `true` and `identityCertificateName` will show the name of the imported identity certificate. For more information about setting up SAML logins, see [Configure security settings](https://enterprise-k8s.arcgis.com/en/latest/administer/configure-security.htm). You can modify the properties returned by this resource using the `update` operation.

<Note type={"note"}>

To configure SAML logins, the [imported SAML identity certificate](/import-identity-certificates) must be named samlcert.

</Note>
