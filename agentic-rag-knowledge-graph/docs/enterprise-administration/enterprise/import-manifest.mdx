## Description

The `importManifest` operation allows organization administrators to import the version manifest into a disconnected environment that can be used to [discover available updates and releases](/check-available-updates) and [upgrade an ArcGIS Enterprise on Kubernetes deployment](/upgrade). The version manifest must be downloaded from [My Esri](https://my.esri.com/), which requires an initial internet connection.


## Request parameters

<StyledTable headers={"Parameter,Details"}>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `manifestFile` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The file containing the version manifest (`.dat` file), used to discover available updates or releases for an ArcGIS Enterprise on Kubernetes deployment.

    </StyledTableCell>
  </StyledTableRow>
  <StyledTableRow>
    <StyledTableCell cellStyle="markdown">

      `f` 

    </StyledTableCell>
    <StyledTableCell cellStyle="markdown">

      The response format. The default format is `html`.

      Values: `html` | `json` | `pjson`

    </StyledTableCell>
  </StyledTableRow>
</StyledTable>
