## Description

The licenses resource lists the current license level of  ArcGIS Mission Server and all authorized extensions. Contact Esri Customer Service if you have questions about license levels or expiration properties.

<StyledTableCell cellStyle="markdown">

</StyledTableRow>

</StyledTable>

## Example usage

Below is a sample request URL for the `properties`  resource:

```javascript
https://machine.domain.com/webadaptor/admin/system/licenses?f=json
```
