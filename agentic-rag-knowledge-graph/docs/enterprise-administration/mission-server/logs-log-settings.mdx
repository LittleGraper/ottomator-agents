## Description

This resource returns the current log settings.

##  Request parameters

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default is `html` .

Values: `html`  \| `json`  \| `pjson` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

Below is a sample request URL for the `settings`  resource:

```javascript
https://machine.domain.com/webadaptor/admin/logs/settings?f=json
```

 

## JSON Response example

```javascript
{
  "settings": {
    "logDir": "C:\\arcgisserver\\logs\\"
    "logLevel": "INFO",
    "maxLogFileAge": 90,
    "maxErrorReportsCount": 10
  }
}
```
