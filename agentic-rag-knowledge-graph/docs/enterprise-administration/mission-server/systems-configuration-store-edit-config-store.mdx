## Description

You can use this operation to update the configuration store. Typically, this operation is used to change the location of the store. 

When  ArcGIS Mission Server is installed, the default configuration store uses local paths.

<Note type={"caution"}>

This operation is a time-consuming operation and causes all server machines to be restarted.

</Note>

##  Request parameters

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`type` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Type of the configuration store.

Values: `FILESYSTEM` 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`connectionString` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

A file path or connection URL to the physical location of the store.

Example

```javascript
connectionString=C:\arcgis\arcgisserver\config-store
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`localRepositoryPath` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

A file path or connection URL to the physical location of the local repository for when the site is in read-only mode.

Example

```javascript
localRepositoryPath=C:\arcgisserver\local
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`move` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

A boolean to indicate if you want to move the content of the current store to the new store. The default is `true` .

Values: `true`  \| `false` 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`runAsync` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Decides if this operation must run asynchronously.

Values: `true`  \| `false` 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is html.

Values: `html`  \| `json` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

Below is a sample POST request `edit` :

```javascript
POST /webadaptor/admin/configstore/edit HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

type=FILESYSTEM&connectionString=C:\arcgis\arcgisserver\config-store&localRepositoryPath=C:\arcgisserver\local&move=true&runAsync=true&f=json

```

 

## JSON Response example

```javascript
{"status": "success"}
```
