## Description

The security resource is a container for all resources and operations that deal with security for your site. Under this resource, you will find resources that represent the users (L) and roles (L) in your current security configuration. Since the content sent to and from this resource (and operations within it) could contain confidential data like passwords, it is recommended that this resource be accessed over HTTPS protocol.
