## Description

##  Request Parameters 

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`serverRole` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Specifies whether the server is a standalone server or a federated server.

Values: `FEDERATED_SERVER`  \| `STANDALONE_SERVER` 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`serverFunction` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Associates a specific function with the server. The only valid server function is `MissionServer` .

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response value is `html` .

Values: `html`  \| `json`  \| `pjson` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

Below is a sample POST request for `update` :

```javascript
POST /webadaptor/admin/security/config/FfURK2ysFiMjzDkJ/update HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

serverRole=FEDERATED_SERVER&serverFunction=MissionServer&f=json
```
