## Description

<Note type={"caution"}>

This operation deletes all the logs on all server machines in the site. This is an irreversible operation.

</Note>

This operation forces the server to clean the logs, and has the effect of freeing up disk space. However, it is not required that you invoke this operation because the server periodically purges old logs.
