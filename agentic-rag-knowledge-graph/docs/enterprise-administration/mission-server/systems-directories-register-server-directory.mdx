## Description

Registers a new server directory. While registering the server directory, you can also specify the directory's cleanup parameters.

<Note type={"note"}>

 You can also register a directory by using its [JSON](/systems-directories) representation as a value of the `directory`  parameter.

</Note>

##  Request parameters

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`name` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The name of the server directory.

Example

```javascript
name=Test Directory
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`physicalPath` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The absolute physical path of the server directory.

Example

```javascript
physicalPath=C:/arcgis/arcgisserver/directories/testDirectory
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`diretoryType` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The type of server directory.

Example

```javascript
directoryType=OUTPUT
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`cleanupMode` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Defines if files in the server directory needs to be cleaned up. The default value is `NONE` .

Example

```javascript
cleanupMode=TIME_ELAPSED_SINCE_LAST_MODIFIED
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`maxFileAge` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

Defines how long a file in the directory needs to be kept before it is deleted.

Example

```javascript
maxFileAge=10
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`description` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

An optional description for the server directory.

Example

```javascript
description=Stores various information generated by services, such as map images
```

 

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is `html` .

Values: `html`  \| `json` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

Below is a sample POST request for `register` :

```javascript
POST /webadaptor/admin/system/directories/register HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

name=Test Directory&physicalPath=C:/arcgis/arcgisserver/directories/testDirectory&directoryType=OUTPUT&description=Stores various information generated by services, such as map images&cleanupMode=TIME_ELAPSED_SINCE_LAST_MODIFIED&maxFileAge=10&f=json
```

 

## JSON Response example

```javascript
{"status": "success"}
```
