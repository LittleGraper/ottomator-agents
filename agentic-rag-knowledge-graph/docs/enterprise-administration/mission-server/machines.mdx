## Description

This resource represents a collection of all the server machines that have been registered with the site; that is, it represents the total computing power of your site. A site will continue to run as long as there is one server machine online.

## Example usage

Below is a sample request URL for the `machines`  resource:

```javascript
https://machine.domain.com/webadaptor/admin/machines?f=json
```
