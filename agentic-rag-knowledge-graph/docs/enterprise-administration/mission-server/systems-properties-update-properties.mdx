## Description

This operation allows you to update the server properties.

<Note type={"note"}>

The update operation overwrites all existing properties.

</Note>

## Example usage

This operation returns information for `AuthTokenTimeInSeconds` , `WebContextURL`  and `WebSocketContextURL` . For more information on these operations, see [Server Properties](/systems-properties).

Below is a sample POST request for `update` , formatted for readability:

```javascript
POST /webadaptor/admin/system/properties/update HTTP/1.1
Host: machine.domain.com
Content-Type: application/x-www-form-urlencoded
Content-Length: []

properties={
  "AuthTokenTimeInSeconds": 180,
  "WebContextURL": "https://mycompany.com/mission",
		"WebSocketContextURL": "wss://mycompany.com/mission"
}&f=json
```

{"status": "success"}
