## Description

A collection of all the [server directories](/systems-directories-directory) is listed under this resource. You can add a new directory using the [Register Directory](/systems-directories-register-server-directory) operation. You can then configure GIS services to use one or more of these directories. If you no longer need the server directory, you must remove the directory by using the [Unregister Directory](/systems-directories-directory-unregister-server-directory) operation.

## Example usage

Below is a sample request URL for the `directories`  resource:

```javascript
https://machine.domain.com/webadaptor/admin/system/directories?f=json
```

 

## JSON Response syntax

```javascript
{
  "directories": [<dir1>, <dir2>]
}
```
