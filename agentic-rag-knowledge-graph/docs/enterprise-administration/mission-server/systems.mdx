## Description

The System resource is a collection of miscellaneous server-wide resources such as [server properties](/systems-properties), [server directories](/systems-directories), the [configuration store](/systems-configuration-store), [Web Adaptors ](/systems-web-adaptors), and [licenses](/systems-licenses).

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is `html` .

Values: `html`  \| `json`

</StyledTableRow>

</StyledTable>

## Example usage

Below is a sample request URL for the `system`  resource:

```javascript
https://machine.domain.com/webadaptor/admin/system?f=json
```

{
