## Description

A read-only resource that returns meta information about the server.

##  Request parameters

<StyledTable headers={"Parameter,Description"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is `html` .

Values: `html`  \| `json` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

Below is a sample request URL for the `info`  resource:

```javascript
https://machine.domain.com/webadaptor/admin/info?f=json
```
