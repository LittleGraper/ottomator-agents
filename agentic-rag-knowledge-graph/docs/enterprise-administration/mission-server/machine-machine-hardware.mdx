## Description

This resource displays hardware information for the specified machine in your  ArcGIS Mission Server site. It updates the information when it detects any change to the configuration of your machine, as well as each time the machine is restarted.

##  Request parameters

<StyledTable headers={"Parameter,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`f` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The response format. The default response format is `html` .

Values: `html`  \| `json` 

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Response properties

<StyledTable headers={"Property,Details"}>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`CPU` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The specifications of the machine's central processing unit (CPU).

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`OS` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The current operating system of the machine.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`systemMemoryMB` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The amount of system memory in megabytes.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`numPhysicalProcessors` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The number of physical cores in the CPU.

</StyledTableCell>

</StyledTableRow>

<StyledTableRow>

<StyledTableCell cellStyle="markdown">

`numLogicalProcessors` 

</StyledTableCell>

<StyledTableCell cellStyle="markdown">

The number of logical cores in the CPU. This value is the number of physical cores multiplied by the number of threads each core can run via hyperthreading.

</StyledTableCell>

</StyledTableRow>

</StyledTable>

## Example usage

Below is a sample request URL for the `hardware`  resource:

```javascript
https://machine.domain.com/webadaptor/admin/machines/MACHINE.DOMAIN.COM/hardware?f=json
```

 

## JSON Response syntax

```other
{
  "cpu": "<Specification of CPU>",
  "os": "<Operating system of machine>",
  "systemMemoryMB": <long integer>,
  "numPhysicalProcessors": <integer>,
  "numLogicalProcessors": <integer>
}
```

 

## JSON Response example

```other
{
  "cpu": "Intel(R) Xeon(R) CPU E5-2698 v3 @ 3.50GHz",
  "os": "Microsoft Windows 10 build 10586",
  "systemMemoryMB": 16306,
  "numPhysicalProcessors": 4,
  "numLogicalProcessors": 8
}
```
